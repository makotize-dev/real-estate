## 都市計画法の全体像

### ◆ 目的（1条）

- 都市の健全な発展と秩序ある整備を図り、国土の均衡ある発展と公共の福祉の増進に寄与する
- 手段：「計画」を立てて、それに沿って土地利用を規制する

### ◆ 規制の入口は「区域指定」

日本中すべての土地が同じように規制されるわけではない。3区分：

| 区分 | 内容 | 規制の強さ |
|---|---|---|
| **都市計画区域** | 一体の都市として総合的に整備・開発・保全する必要がある区域 | 強 |
| **準都市計画区域** | 都市計画区域**外**で、相当数の建築物等の建築・造成が現に行われ／見込まれる区域 | 中（限定的） |
| それ以外 | 上記いずれにも指定されない区域 | 都市計画法の規制対象外 |

## 4条：用語の定義（開発行為・特定工作物）

### ◆ 4条12項：開発行為の定義

> 「主として**建築物の建築**又は**特定工作物の建設**の用に供する目的で行なう**土地の区画形質の変更**」

→ 2要素で構成：
1. 主として建築物の建築 or 特定工作物の建設の用に供する目的
2. 土地の区画形質の変更を伴う

### ◆ 4条11項：特定工作物の定義

| 区分 | 内容 | 例 |
|---|---|---|
| **第一種特定工作物** | 周辺環境を**悪化させるおそれがある工作物**（政令で指定） | コンクリートプラント、アスファルトプラント等 |
| **第二種特定工作物** | **大規模な工作物**（政令で指定） | ゴルフコース、運動・レジャー施設、墓園等 |

### ◆ 「区画形質の変更」の3要素（行政解釈）

**出典**：国土交通省「**開発許可制度運用指針**」（平成26年8月1日 国都計第67号、最終改正 令和7年3月19日 国都計第193号）。地方自治法245条の4に基づく技術的助言

| 要素 | 意味 |
|---|---|
| **区画** | 敷地を分割・統合して形を変えること |
| **形** | 切土・盛土などで高さを変えること |
| **質** | 用途を変更すること（例：農地→宅地） |

→ 3要素のいずれかに該当すれば「区画形質の変更」あり

## 5条：都市計画区域の指定

### ◆ 指定権者【頻出】

| 場面 | 指定権者 |
|---|---|
| 原則 | **都道府県** |
| 2以上の都府県にまたがる場合 | **国土交通大臣**（関係都府県の意見聴取） |

→ ひっかけ：「2以上にまたがる場合は知事の協議で決める」は誤り

### ◆ 指定の対象

- 市又は政令で定める要件に該当する町村の中心の市街地を含む区域
- 自然的・社会的条件、人口・土地利用・交通量等を勘案して、一体の都市として総合的に整備等する必要がある区域

---

## 5条の2：準都市計画区域

### ◆ 指定権者・対象

- 指定権者：**都道府県**
- 対象：都市計画区域**外**で、相当数の建築物等の建築・造成が現に行われ／見込まれる区域、放置すれば将来の都市整備に支障が生じるおそれがある区域

### ◆ 趣旨

都市計画区域外でも、開発が進みつつある場所を放置すると将来困る → 先回りして規制をかける

## 7条：区域区分（線引き）

### ◆ 区域区分の任意/義務【頻出】

| 場面 | 扱い | 根拠 |
|---|---|---|
| 原則 | 区域区分を定めることが**できる**（任意） | 7条1項本文 |
| 三大都市圏の特定地域・大都市に係る都市計画区域として政令で定めるもの（政令指定都市等） | 区域区分を定める**ものとする**（義務） | 7条1項但書 |

→ ひっかけ：「すべての都市計画区域で区域区分が必要」は誤り

### ◆ 義務となる三大都市圏（7条1項1号）

- イ：首都圏整備法に基づく既成市街地・近郊整備地帯
- ロ：近畿圏整備法に基づく既成都市区域・近郊整備区域
- ハ：中部圏開発整備法に基づく都市整備区域

### ◆ 市街化区域・市街化調整区域の定義

| 区分 | 定義 |
|---|---|
| **市街化区域**（7条2項） | 既に市街地を形成している区域 + おおむね**10年以内**に優先的かつ計画的に市街化を図るべき区域 |
| **市街化調整区域**（7条3項） | 市街化を**抑制**すべき区域 |

### ◆ 「区域区分が定められていない都市計画区域」（俗称：未線引き・非線引き）

- 法律上の用語は「区域区分が定められていない都市計画区域」（条文に「未線引き」「非線引き」の語はない）
- 規制対象外ではない：用途地域・開発許可等の規制はかかりうる
- 「規制対象外」になるのは都市計画区域・準都市計画区域いずれにも指定されていない区域

## 8〜9条：地域地区・用途地域

### ◆ 用途地域とは

土地の利用方法を制限する地域地区。**建ててもいい建物・建ててはいけない建物**を決める

### ◆ 用途地域13種類（9条）【頻出】

| グループ | 種類数 | 用途地域 |
|---|---:|---|
| **住居系** | 8 | 1.第一種低層住居専用 / 2.第二種低層住居専用 / 3.第一種中高層住居専用 / 4.第二種中高層住居専用 / 5.第一種住居 / 6.第二種住居 / 7.準住居 / 8.田園住居 |
| **商業系** | 2 | 9.近隣商業 / 10.商業 |
| **工業系** | 3 | 11.準工業 / 12.工業 / 13.工業専用 |

### ◆ 性格の方向性

- 住居系（特に「専用」がつく地域）：**住環境保護が最も厳しい**、商業・工業施設は厳しく制限
- 商業系：商業・業務の利便を優先、住居も可能
- 工業系（特に「工業専用」）：**工業利便を優先**、工業専用地域は**住宅も建てられない**

→ 住居系 → 商業系 → 工業系 の順に、住環境保護が弱まる

### ◆ 住居系8種類のサブグループ

| サブグループ | 含まれる用途地域 | 特徴 |
|---|---|---|
| 低層住居専用 | 第一種・第二種 | 一戸建て中心、3階建て程度まで |
| 中高層住居専用 | 第一種・第二種 | マンション可、用途制限は厳しめ |
| 住居 | 第一種・第二種・準住居 | 「専用」がつかない、店舗・事務所もある程度可 |
| 田園住居 | 田園住居【2018年新設】 | 農業利便と住居環境の調和 |

→ 住居系内：**第一種＝厳しい／第二種＝緩い／準＝最も緩い**

## 13条1項7号：用途地域の定め方【頻出】

| 区域 | 用途地域の扱い | 条文文言 |
|---|---|---|
| **市街化区域** | **必ず定める** | 「少なくとも用途地域を定めるものとし」 |
| **市街化調整区域** | **原則として定めない**（例外あり） | 「原則として用途地域を定めないものとする」 |
| 区域区分が定められていない都市計画区域 | 13条1項7号には明示規定なし | — |
| 準都市計画区域 | 別規定（13条3項） | — |

### ◆ 論理

- 市街化区域：市街化を進める → 用途整理が必要
- 市街化調整区域：市街化を抑制する → 積極的な土地利用を想定しない → 用途地域は原則不要

## 29条：開発許可【頻出】

### ◆ 制度の趣旨

無秩序な市街化を防止し、計画的な市街化を誘導するため、一定の開発行為に**事前許可**を必要とする制度

### ◆ 許可権者

**都道府県知事**（指定都市・中核市等の区域内では当該指定都市等の長）

### ◆ 許可対象（29条1項：都市計画区域・準都市計画区域内）

開発行為（4条12項）をしようとする者は、原則として許可必要。ただし規模要件あり。

### ◆ 規模要件【頻出】（施行令19条）

| 区域 | 許可が必要な開発行為の規模 |
|---|---|
| **市街化区域** | **1,000㎡以上** |
| **市街化調整区域** | **規模を問わず許可必要**（面積要件なし） |
| **区域区分が定められていない都市計画区域**（非線引き） | **3,000㎡以上** |
| **準都市計画区域** | **3,000㎡以上** |
| **都市計画区域・準都市計画区域外** | **10,000㎡（1ha）以上** |

→ 市街化調整区域の**面積要件なし**が最重要。市街化抑制区域なので小規模開発も規制

### ◆ 条例による引下げ

都道府県・指定都市等の条例で、開発許可を要する規模を**300㎡まで**引き下げ可能

### ◆ 「開発行為」に該当する典型例

| 例 | なぜ該当するか |
|---|---|
| 農地を切土・盛土して住宅地を造成し戸建て分譲 | 区画形質変更＋建築物の建築目的 |
| 山林を伐採・整地してマンション用地にする | 形質変更＋建築物の建築目的 |
| 既存宅地を分筆・道路新設して建売分譲 | 区画変更＋建築物の建築目的 |
| 山林を切り開いてゴルフコース造成 | 形質変更＋第二種特定工作物の建設目的 |
| コンクリートプラント設置のため造成 | 形質変更＋第一種特定工作物の建設目的 |

### ◆ 「開発行為」に該当しない例【ひっかけ】

| 例 | なぜ該当しないか |
|---|---|
| 既存宅地に新築するだけ（造成等を伴わない） | 区画形質変更がない |
| 農地を青空駐車場にする（建物なし） | 建築物・特定工作物の建設目的でない |
| 山林を伐採して資材置場にする（建物なし） | 建築物・特定工作物の建設目的でない |
| 単なる分筆（土地登記上の分割のみ） | 物理的な区画形質変更がない |

→ 出典：宅建参考書・解説サイトの典型的説明（行政解釈の正確な原典は開発許可制度運用指針を要確認）

### ◆ 判定フロー

```
1. 当該行為は「開発行為」か？
   - 主として建築物の建築 or 特定工作物の建設の用に供する目的か
   - 土地の区画形質の変更を伴うか
   → Yes なら 開発行為

2. 開発行為の場所・規模は？
   - 市街化区域：1,000㎡以上 → 許可必要
   - 市街化調整区域：規模問わず → 許可必要
   - 非線引き・準都市計画区域：3,000㎡以上 → 許可必要
   - 都市計画区域外：10,000㎡以上 → 許可必要

3. 許可不要のケース（29条1項各号）に該当するか？
   - 詳細は下記「29条1項各号」セクション参照
```

## 29条1項各号：許可不要の例外（11号）【頻出】

### ◆ 11号の分類と立法趣旨

| 号 | 内容 | 立法趣旨カテゴリ |
|---|---|---|
| **1号** | 規模未満（市街化区域・非線引・準都計のみ） | 規模軽微 |
| **2号** | 農林漁業用建築物（市街化調整区域・非線引・準都計のみ） | 地域産業の保護 |
| **3号** | 駅舎・図書館・公民館・変電所等の**広域**公益建築物 | 公益施設 |
| **4号** | 都市計画事業の施行 | 他法令で代替審査済み |
| **5号** | 土地区画整理事業の施行 | 同 |
| **6号** | 市街地再開発事業の施行 | 同 |
| **7号** | 住宅街区整備事業の施行 | 同 |
| **8号** | 防災街区整備事業の施行 | 同 |
| **9号** | 公有水面埋立で告示前 | 別法（公水法）で管理中 |
| **10号** | 非常災害の応急措置 | 緊急性 |
| **11号** | 通常の管理行為・軽易な行為 | 規模軽微・規制になじまない |

### ◆ 区域別の例外適用範囲【頻出】

| 区域 | 「規模軽微」例外（1号） | 「農林漁業」例外（2号） | 3〜11号 |
|---|:---:|:---:|:---:|
| 市街化区域 | ✅ 使える | ❌ 使えない | ✅ 使える |
| 市街化調整区域 | ❌ **使えない** | ✅ 使える | ✅ 使える |
| 非線引・準都計 | ✅ 使える | ✅ 使える | ✅ 使える |

→ 市街化調整区域では「**規模が小さくても許可必要**」というのが重要。
→ 「市街化区域は規模で逃げる／市街化調整区域は農林漁業で逃げる」で覚える。

### ◆ 4〜9号の市街化調整区域での実質的適用可能性

条文上は4〜9号いずれも区域限定なしだが、各事業の制度的性質から：

| 号 | 実質的可能性 | 補足 |
|---|---|---|
| 4号 都市計画事業 | **ありうる** | 市街化調整区域を通過する都市計画道路など |
| 5号 土地区画整理事業 | ありうる（実例少） | 「市街化抑制」と矛盾しうるので慎重判断 |
| 6号 市街地再開発事業 | 事実上なし | 「市街地」が前提 |
| 7号 住宅街区整備事業 | 事実上なし | 大都市住宅供給制度 |
| 8号 防災街区整備事業 | 事実上なし | 密集市街地が前提 |
| 9号 公有水面埋立 | ほぼなし | 市街化調整区域指定の公水面は内陸湖沼の極稀例外 |

→ 試験では**条文上の区域限定の有無で機械的に判断**する（実質可能性まで問われることは稀）

### ◆ 試験ひっかけパターン

- 「市街化調整区域で小規模な住宅開発 → 規模未満で許可不要」→ **誤り**（1号は市街化調整区域で使えない）
- 「市街化区域で農家の自宅建築 → 農林漁業用で許可不要」→ **誤り**（2号は市街化区域で使えない）
- 「市街化調整区域で駅舎建築 → 開発許可必要」→ **誤り**（29条1項3号で許可不要、区域限定なし）

## 33条：開発許可の基準（技術基準）【頻出】

### ◆ 羈束行為であること【最重要】

33条1項本文：「都道府県知事は…許可を**しなければならない**」

→ **裁量ではない**。基準を満たせば必ず許可、満たさなければ必ず不許可。

### ◆ 試験のひっかけ

- 「環境上好ましくないので許可しない」「住民が反対しているので不許可」→ **誤り**（裁量判断できない）
- 「基準を満たしても許可しないことができる」→ **誤り**（羈束行為）

### ◆ 14号のグループ分け

| グループ | 号 | 何を見るか |
|---|---|---|
| **A 計画適合** | 1号・5号 | 用途地域・地区計画への適合 |
| **B 公共施設・インフラ** | 2号・3号・4号・6号 | 道路・公園・排水・給水・学校等 |
| **C 安全・環境** | 7号・8号・9号・10号 | 擁壁・排水安全／災害危険区域除外／樹木・表土保全／緩衝帯 |
| **D 利便性** | 11号 | 輸送便利性 |
| **E 申請者の適格性** | 12号・13号・14号 | 資力・信用／工事能力／関係権利者の同意 |

### ◆ 自己居住用住宅の特例

33条1項では **12〜14号の基準**（資力・信用／工事能力／関係権利者の同意）について、「**政令で定める場合に限り適用する**」となっており、自己居住用住宅（マイホーム）には**適用されない**運用。

→ 試験：「マイホームの開発許可で資力・信用の証明を求められる」→ **誤り**

※ 正確な対象範囲は施行令で規定（要確認事項）

### ◆ 条例による技術基準の強化

地方公共団体は条例で技術的細目を**強化・付加**できる。地域実情に応じた上乗せ規制が可能。

出典：e-Gov 都市計画法33条（一次確認済み）

## 34条：市街化調整区域における特別基準【頻出】

### ◆ 33条との関係【最重要】

市街化調整区域での開発許可は：

> **33条の技術基準（共通）をクリア** ＋ **34条のいずれかの号に該当**

→ **両方** が必要。市街化区域より審査が厳しい二段階構造。

### ◆ 14号 + 8の2号の論点グループ

| グループ | 号 | 立法趣旨 |
|---|---|---|
| **A 立地必然性** | 2号（鉱物・観光・第一種特定工作物）・4号（農林漁業＋加工処理） | その場所でしかできない |
| **B 公益・地域必要性** | 1号（公益建築物・日常店舗）・3号（特殊条件事業）・5号（特定農山村活性化）・6号（中小企業連携）・7号（既存工場関連）・8号（危険物施設） | 公益上の必要 |
| **C 既存権利の保護** | 8の2号（災害区域からの代替）・11号（市街化区域隣接の連たん地区）・13号（区域区分決定時の既権利者） | 既得権・救済 |
| **D 条例・計画・審査会** | 10号（地区計画適合）・12号（条例限定）・**14号**（開発審査会の議） | 個別判断の余地 |
| **E その他** | 9号（その他政令で定める） | 補充 |

### ◆ 34条14号【最頻出】

> 「**開発審査会の議を経て**、**開発区域の周辺における市街化を促進するおそれがなく**、かつ、**市街化区域内において行うことが困難又は著しく不適当**と認める開発行為」

**3要件すべて**が必要：

| 要件 | 内容 |
|---|---|
| ① 手続要件 | 開発審査会の議を経ること |
| ② 実体要件1 | 周辺の市街化促進のおそれがない |
| ③ 実体要件2 | 市街化区域でやることが困難または著しく不適当 |

→ 「審査会の議を経るだけ」で許可されるわけではない（実体要件②③も必要）

### ◆ 34条14号の機能

- 1〜13号：列挙された定型ケース
- **14号**：列挙からこぼれた非定型ケースを、審査会の慎重審議＋実体要件で**例外的に許可する道**
- 「許可しにくくする」のではなく「**1〜13号以外でも個別判断で許可する余地を与える**」規定。ただし安易に許可されないよう手続要件と実体要件で歯止め

出典：e-Gov 都市計画法34条（一次確認済み）

## 29条1項 vs 34条 混同整理【頻出】

| 同じテーマ | 29条1項（許可**不要**） | 34条（市街化調整区域：許可は**必要**だが認められる） | 区別の決め手 |
|---|---|---|---|
| **公益建築物** | **1項3号**：駅舎・図書館・公民館・変電所など**広域公益施設** | **1号**：周辺住民の**日常生活**に必要な店舗・診療所等の**地域密着型施設** | 広域 vs 地域密着 |
| **農林漁業用** | **1項2号**：施行令で定める**限定的なもの**（畜舎・農作業舎・農業者の自宅等） | **4号**：1項2号の政令で定めるもの**以外**＋**加工処理施設** | 典型 vs 加工処理 |

### ◆ 整理の核心

> **29条1項に該当する**：許可手続そのものが不要
> **29条1項に該当しない**：33条＋（市街化調整区域なら）34条で審査

→ 「許可不要」と「許可されやすい」は**別の話**

### ◆ 試験ひっかけパターン

| 問 | 結論 | 根拠 |
|---|---|---|
| 市街化調整区域で農産物加工施設は29条1項2号で許可不要 | **誤り** | 加工施設は1項2号の政令対象外。34条4号で許可必要 |
| 市街化調整区域で駅舎建築は許可必要 | **誤り** | 29条1項3号で許可不要（区域限定なし） |
| 開発審査会の議だけで34条1〜13号非該当でも必ず許可 | **誤り** | 14号は3要件構造（手続＋実体2要件） |

## 都計法36〜43：開発許可後の手続・建築制限【頻出・R1総点検6/28】

### ◆ 37条 公告"前"の建築制限
- **原則：建築物・特定工作物は一切建築禁止**（×「公共施設以外」）。
- 例外：①工事用仮設・知事が支障なし認定／②33条1項14号の不同意者の権利行使。

### ◆ 42条 公告"後"の建築制限
- **原則：予定建築物等以外は新築・新設禁止**（×「何でも建てられる」）。
- 例外：知事が支障なし認定で許可／用途地域等が定められているとき（→制限が外れる）。

### ◆ 41条 用途地域の定めのない区域
- 知事が建蔽率・高さ・壁面の位置等を**定めることができる**（用途地域の代替規制）。

- 市街化調整区域の**二重関門**：開発許可(29＝造成)＋建築許可(43＝建築)。

### ◆ 建築基準法の2本柱【最重要】

| | 単体規定 | 集団規定 |
|---|---|---|
| 規律対象 | **建物そのもの**の安全・衛生 | 建物と**周辺環境・街**との関係 |

### ◆ なぜ集団規定は「都市計画区域・準都市計画区域内」だけか

- 単体規定：建物にいる人の安全 → どこの建物でも必要 → 全国適用

→ 都市計画法の「区域指定」と建築基準法の集団規定の適用範囲がリンク。**都計法と建基法はセットで機能する**。

## 42条：道路の定義【頻出】

出典：建築基準法42条（e-Gov 一次確認済み）

### ◆ 1項：建築基準法上の「道路」とは

「道路」＝次の1〜5号のいずれかに該当する、**幅員4m以上**（特定行政庁が都道府県都市計画審議会の議を経て指定する区域内では**6m以上**）のもの（地下のものを除く）。

| 号 | 内容 | 道の存否 |
|---|---|---|
| 1号 | 道路法による道路（公道） | 指定等の時点で**すでに存在** |
| 2号 | 都市計画法・土地区画整理法・都市再開発法等による道路 | すでに存在 |
| 3号 | 都市計画区域等の指定等によりこの章の規定が適用されるに至った際、現に存在する道 | すでに存在 |
| 4号 | 新設・変更の事業計画があり、2年以内に事業執行予定として特定行政庁が指定した道路（計画道路） | 指定の時点では**まだ存在しない**（これから築造） |
| 5号 | 政令基準に適合する道を築造しようとする者が特定行政庁から位置の指定を受けたもの（位置指定道路） | まだ存在しない（これから築造） |

→ 区別の軸は「**指定等の時点でその道が物理的に存在しているか**」。1〜3号＝既存の道／4・5号＝これから築造される道。

### ◆ 2項：みなし道路（実務上「2項道路」「みなし道路」。条文の語ではない）

- **集団規定（この章の規定）が適用されるに至った時点で、すでに建築物が立ち並んでいた**幅員4m未満の道で、特定行政庁が指定したもの → 1項の道路と**みなす**
- 道路境界線：その道の**中心線から水平距離2m**（指定区域内は3m。特定行政庁が支障なしと認める場合は2m）の線を境界線とみなす
- **セットバック**：建て替え時、この境界線まで敷地を後退させる。後退部分は建築不可（道路内建築の制限）・敷地面積に算入不可。ただし**所有権は失わない**
- 中心線基準＝両側が2mずつ折半で負担。両側がセットバックすれば最終的に道幅4m
- **ただし書**：道が中心線から2m未満で崖地・川・線路敷地等に沿う場合 → 崖地等の道の側の境界線、および**そこから道の側に水平距離4m**の線を境界線とみなす（後退できない側の分も建築可能側が負担）

### ◆ 2項の趣旨：既成事実の救済

建築基準法の「幅員4m」ルールが後から適用されたとき、4m未満の道を「道路でない」と切り捨てると沿道の既存の家がすべて接道義務違反になる。**ルール適用前から建ち並んでいた街並み**を道路として救済するのが2項。だから「適用されるに至った際、現に建ち並んでいた」ことが要件。

## 43条：敷地等と道路との関係（接道義務）【頻出】

出典：建築基準法43条（e-Gov 一次確認済み）

### ◆ 趣旨

接道義務の目的は**避難**（人が安全に逃げる）と**緊急車両のアクセス**（消防・救急）の確保。3項に「避難又は通行の安全」の文言あり。

### ◆ 1項：接道義務

- **建築物の敷地は、道路に2m以上接しなければならない**
- ただし、接する相手の「道路」から次を除く：
  - 1号　自動車のみの交通の用に供する道路（自動車専用道路）
  - 2号　一定の地区計画の区域内の道路
- 自動車専用道路が除外される理由：沿道の敷地と遮断され出入口がない＝人も緊急車両も道路から敷地に出入りできず、接道義務の趣旨を満たさない

### ◆ 2項：接道義務の例外（一定の建築物に1項を適用しない）

条文柱書：「前項の規定は、次の各号のいずれかに該当する**建築物**については、**適用しない**」。
※例外の名宛人は「敷地」ではなく「**建築物**」。1号が建築物の用途・規模を要件にするため。

| 号 | 内容 | 建築審査会の同意 | 呼称 |
|---|---|:---:|---|
| 1号 | 敷地が幅員4m以上の「道」（**道路に該当するものを除く**・避難通行の安全の省令基準適合）に2m以上接し、利用者が少数の用途・規模の建築物で、特定行政庁が支障なしと認めるもの | **不要** | 認定（実務上の通称） |
| 2号 | 敷地の周囲に広い空地を有する等の建築物で、特定行政庁が建築審査会の同意を得て**許可**したもの | **必要** | 許可（条文の語） |

→ 試験ポイント：**建築審査会の同意が必要なのは2号の許可だけ**。1号には不要。（建築審査会の組織は **78〜80条** 参照）

### ◆ 2項の趣旨

接道義務（1項）は「42条の道路に2m接する」という**形式要件**。本来の目的（避難・通行・防火・衛生の安全）が、その形式によらず実質的に確保されている建築物——1号＝4m以上の安全基準適合の道に接する小規模・少人数の建築物／2号＝周囲に広い空地——には、形式的義務を課さない。

### ◆ 3項：条例による付加（上乗せ条例）

- 地方公共団体は、次の5類型の建築物について、1項では避難・通行の安全の目的を十分に達成できないと認めるとき、**条例で**接道義務を**加重（付加）できる**
  - ①特殊建築物　②階数3以上　③政令で定める無窓居室を有する建築物　④延べ面積1,000㎡超　⑤袋路状道路にのみ接し延べ面積150㎡超（一戸建ての住宅を除く）
- できるのは**付加（厳しく）のみ**。緩和は不可（接道義務は全国一律の最低基準）

## 52条：容積率【最頻出】

出典：建築基準法52条（e-Gov 一次確認済み・全15項）

### ◆ 容積率の意義

- **容積率 = 延べ面積 ÷ 敷地面積**
- 5/27 学習の建蔽率（53条）と並ぶ「形態制限」の柱
- 立法趣旨：人口密度・インフラ容量（上下水道・道路・電気）の制御
- 高さ制限とは別物（容積率は延べ床面積を制限・高さは別途55・56・56条の2 で規制）

### ◆ 1項：用途地域別の容積率上限【最頻出】

| 号 | 地域 | 容積率（10分の○）|
|----|------|--------------|
| 一 | 1低・2低・田園住居 | 5・6・8・10・15・20 |
| 二 | 1中高・2中高・1住・2住・準住居・近商・準工 | 10・15・20・30・40・50 |
| 三 | 商業地域 | 20・30・40・50・60・70・80・90・100・110・120・130（11段階）|
| 四 | 工業・工業専用 | 10・15・20・30・40 |
| 五 | 高層住居誘導地区 | 1項2号の数値〜1.5倍の範囲 |
| 六 | 居住環境向上用途誘導地区 | 都市計画で定められた数値 |
| 七 | 特定用途誘導地区 | 都市計画で定められた数値 |
| 八 | 用途地域指定なし区域 | 5・8・10・20・30・40（**特定行政庁が都道府県都市計画審議会の議を経て**定める）|

5/27 53条との対比：53条（建蔽率）は商業地域が**8/10で固定**だが、52条（容積率）は商業地域も**段階区分**（細かい都市計画的調整が可能）。

### ◆ 5号 高層住居誘導地区【試験論点】

- 対象は1住・2住・準住居・近隣商業・準工業地域内の都市計画指定地区
- 住宅床面積が**延べ面積の3分の2以上**の建築物が対象
- 容積率は1項2号の数値の**1.5倍以下**の範囲で別途定められる
- 趣旨：都心部の住居供給促進（職住近接の高層住宅誘導）

### ◆ 2項：前面道路幅員制限【最頻出】

> 前面道路の幅員が**12m未満**の場合、容積率は **前面道路幅員 × 係数** 以下でなければならない

| 号 | 地域 | 係数 |
|----|------|------|
| 一 | 1低・2低・田園住居 | **4/10** |
| 二 | 1中高・2中高・1住・2住・準住居（高層住居誘導地区除く）| **4/10**（特定行政庁指定区域では 6/10）|
| 三 | その他（商業・近商・準工・工業・工業専用 等）| **6/10**（特定行政庁指定区域では 4/10 または 8/10）|

**適用ルール**：1項と2項の**両方の制限を満たす必要があり、より厳しい方が適用される**

### ◆ 具体例

第一種住居地域（容積率指定30/10）の敷地、前面道路6m：
- 1項：300%
- 2項：6 × 4/10 = 240%
- **適用**：240%（厳しい方）

### ◆ 3〜6項：延べ面積から除外する部分

容積率の分子（延べ面積）から特定部分を除外する規定群（実質的緩和）：

| 項 | 除外される部分 | 上限 |
|----|--------------|------|
| 3項 | 地階の住宅・老人ホーム等の用途部分（天井が地盤面から1m以下）| 当該住宅等の床面積合計の**1/3** |
| 6項1号 | 政令で定める昇降機（エレベーター）の昇降路 | 上限なし |
| 6項2号 | 共同住宅・老人ホーム等の**共用廊下・階段** | 上限なし |
| 6項3号 | 住宅・老人ホーム等の機械室（給湯設備等）| 上限なし |

### ◆ 趣旨

容積率本来の目的は**人がいる空間の総量**の制御。共用廊下・昇降機・機械室等は「人が常時いる場所」ではないので除外しても本来の目的を損なわない。加えて、バリアフリー化・地下空間活用・共同住宅建設等の**政策誘導**の側面もある。

### ◆ 7項：敷地が地域にまたがる場合（面積按分）

5/27 53条2項と同じ構造：各地域の容積率上限 × その地域の敷地面積割合の合計。

| 規制 | 敷地またがり時の扱い |
|------|------------------|
| 用途制限（48条）| 過半主義 |
| 建蔽率（53条2項）| 面積按分 |
| **容積率（52条7項）** | **面積按分** |
| 防火地域・準防火地域 | 厳しい方を全体に |

### ◆ 8項：住宅含む建築物の1.5倍特例

住居系・近商・準工・商業地域で一定条件（敷地内空地・敷地面積等の政令基準）を満たすと、容積率を1.5倍まで緩和。

### ◆ その他の項

- 4・5項：地盤面の定義（技術的）
- 9項：特定道路（幅員15m以上）接続による緩和
- 10〜13項：壁面線・計画道路による特例
- 14項：特定行政庁の許可による特例（機械室の床面積大・周囲に空地等）
- 15項：44条2項準用（手続）

### ◆ 試験頻出ひっかけ

- 「容積率は用途地域に応じて法律で固定」 ❌（都市計画で複数選択肢から定める）
- 「高層住居誘導地区の容積率緩和は3倍まで」 ❌（**1.5倍まで**）
- 「用途地域指定なし区域は容積率なし」 ❌（特定行政庁が定める）
- 「前面道路が12m以上なら2項は適用なし」 ✅（1項のみで判定）
- 「前面道路が2以上ある場合は幅員の最小のもの」 ❌（**最大のもの**）

## 53条：建蔽率【頻出】

出典：建築基準法53条（e-Gov 一次確認済み・全9項）

### ◆ 建蔽率の意義

- **定義**：建築面積（同一敷地内に2以上の建築物があるときはその合計）／敷地面積
- **立法趣旨**（標準的説明・逐語未確認）：敷地内の空地確保による防火・延焼防止・通風・採光・避難経路・日照の確保
- **容積率（52条）との区別**：建蔽率は水平投影面積の制限（平面）／容積率は延べ床面積の制限（立体ボリューム）／高さ制限（55・56・56条の2）はさらに別

### ◆ 1項：用途地域別の建蔽率上限【最頻出】

| 号 | 地域 | 建蔽率の選択肢 |
|----|------|-------------|
| 一 | 1低・2低・1中高・2中高・田園住居・**工業専用** | 3/10・4/10・5/10・6/10 |
| 二 | 1住・2住・準住居・**準工** | 5/10・6/10・8/10 |
| 三 | 近隣商業 | 6/10・8/10 |
| 四 | **商業地域** | **8/10**（選択肢なし・固定）|
| 五 | 工業 | 5/10・6/10 |
| 六 | 用途地域指定なし | 3/10・4/10・5/10・6/10・7/10 |

**商業地域が8/10固定の意味**：他の地域は自治体に「より厳しくする」裁量があるが、商業地域では8/10を一律で保証する（土地利用効率の最大化）。

### ◆ 2項：敷地が地域にまたがる場合（面積按分）【頻出】

敷地が複数の建蔽率の異なる地域にまたがるとき：
- **各地域の建蔽率 × その地域の敷地面積割合の合計**
- 例：第一種住居(60%)に200㎡＋商業(80%)に100㎡（計300㎡）→ 60%×200/300 + 80%×100/300 = 66.67%

W3で学んだ48条用途制限（過半主義）と対比：
- **用途制限（48条）= 過半主義**（敷地の過半が属する地域の規制を全体に適用）
- **建蔽率（53条2項）・容積率（52条）= 面積按分**
- **防火地域・準防火地域 = 厳しい方を全体に適用**

### ◆ 3項：緩和規定【最頻出】

「**十分の一を加える**」「**十分の二を加える**」── 加算（+1/10、+2/10）であって乗算ではない。

| 該当 | 緩和量 |
|------|------|
| 1号のみ（防火地域＋耐火建築物等／準防火地域＋耐火or準耐火建築物等）| **+1/10** |
| 2号のみ（街区の角地等で特定行政庁が指定）| **+1/10** |
| 1号と2号両方 | **+2/10** |

**重要な但し**：1号は「建蔽率8/10地域を除く」（1項2〜4号の8/10指定区域・商業地域は対象外）。これらの地域は緩和ルートではなく**6項1号の完全不適用**ルートに乗る。

### ◆ 6項：完全不適用【最頻出】

次のいずれかに該当する建築物は建蔽率の制限を**全く受けない**：

| 号 | 内容 | 効果 |
|----|------|------|
| 一 | 8/10地域（商業・近商の8/10指定・1住等の8/10指定）＋**防火地域**内＋**耐火建築物等** | **建蔽率10/10**（敷地100%に建物可）|
| 二 | 巡査派出所・公衆便所・公共用歩廊等 | 性質上不適用 |
| 三 | 公園・広場・道路・川等の内にある建築物（特定行政庁の許可）| 特定行政庁許可で不適用 |

**3項緩和と6項不適用の関係**：
- 通常の地域＋防火地域＋耐火 → 3項1号で **+1/10 緩和**
- 8/10地域＋防火地域＋耐火 → 6項1号で**完全不適用（10/10）**

→ 駅前商業地区で敷地いっぱいに高層ビルが建てられるのはこの規定の効果。

### ◆ 7項・8項：敷地が防火・準防火地域内外にまたがる場合のみなし規定

**7項**：防火地域の内外にまたがる敷地＋建物全部が**耐火建築物等** → 敷地全部を**防火地域内とみなして** 3項1号または6項1号を適用

**8項**：準防火地域とそれ以外の区域にまたがる敷地＋建物全部が**耐火or準耐火建築物等** → 敷地全部を**準防火地域内とみなして** 3項1号を適用

→ 敷地の一部だけ防火・準防火地域でも、建物全部が耐火等なら全体扱いで緩和・不適用を受けられる。

### ◆ 4項・5項：壁面線等を遵守＋特定行政庁の許可で緩和

- 4項：壁面線・壁面位置制限を遵守＋特定行政庁が安全上・防火上・衛生上支障なしと認めて許可 → 緩和
- 5項：特定計画区域・防災街区整備地区等の制限を遵守＋特定行政庁の許可 → 緩和
- いずれも実務上は出題頻度低め。

### ◆ 試験頻出ひっかけ

- 「緩和は基準値に**乗算**」 ❌（**加算** +1/10・+2/10）
- 「商業地域は3項1号で防火＋耐火なら9/10」 ❌（**6項1号で完全不適用＝10/10**）
- 「敷地が2地域にまたがるときは過半主義」 ❌（建蔽率は**面積按分**。過半主義は用途制限）
- 「敷地の一部が防火地域なら全体に防火規制」 ❌（**全部が耐火建築物等**の条件付きで全体みなし）

### ◆ 計算例

| ケース | 計算 | 上限 |
|------|------|------|
| 第一住居（指定60%）の角地＋防火地域内＋耐火建築物等 | 6/10 + 2/10（3項1号＋2号両方）| **8/10** |
| 商業地域＋防火地域＋耐火建築物等 | 6項1号で完全不適用 | **10/10** |
| 第一住居（60%）200㎡＋商業（80%）100㎡＝計300㎡敷地 | 60% × 200/300 + 80% × 100/300 | **66.67%** |

## 55条：第一種低層住居専用地域等内の建築物の高さの限度【頻出】

出典：建築基準法55条（e-Gov 一次確認済み）

### ◆ 制度（絶対高さ制限）

> 「第一種低層住居専用地域、第二種低層住居専用地域又は田園住居地域内においては、建築物の高さは、**十メートル又は十二メートル**のうち当該地域に関する都市計画において定められた建築物の高さの限度を超えてはならない。」

- 対象地域：**1低・2低・田園住居**の3地域**のみ**
- 上限：**10m または 12m**（都市計画で定める）
- 趣旨：低層住居環境の保護（戸建て・低層集合住宅の街並み）

### ◆ 2項：10m地域での緩和

10m と定められた地域でも、敷地内に政令で定める空地を有し、政令規模以上の敷地面積で、特定行政庁が良好な住居環境を害さないと認める建築物は、高さを**12m**まで認める。

### ◆ 3項：再エネ設備設置時の特例

再生可能エネルギー源（太陽光等）の利用設備設置のための工事で構造上やむを得ないもの・特定行政庁の許可で前2項の限度超え可。

### ◆ 4項：不適用となる例外

- 1号：敷地周囲に広い公園・広場・道路等の空地を有し、特定行政庁の許可
- 2号：学校等で用途上やむを得ない・特定行政庁の許可

### ◆ 試験頻出

- 対象は**1低・2低・田園住居のみ**（他地域は55条の適用なし、別途斜線制限等で対応）
- 上限は**10m または 12m**（都市計画で選択）

## 56条：建築物の各部分の高さ（斜線制限）【頻出】

出典：建築基準法56条（e-Gov 一次確認済み）

### ◆ 3類型の斜線制限

| 種類 | 制限の方向 | 適用地域 | 根拠 |
|------|----------|---------|------|
| **道路斜線** | 前面道路の反対側境界線から斜め上向き | ほぼ全用途地域 | 56条1項1号 |
| **隣地斜線** | 隣地境界線から斜め上向き | 1低・2低・田園住居**以外** | 56条1項2号 |
| **北側斜線** | 真北方向の境界線から斜め上向き | 1低・2低・田園住居・1中高・2中高 | 56条1項3号 |

### ◆ 趣旨

- 道路斜線：道路向かいの建物の採光阻害・道路への圧迫感防止
- 隣地斜線：隣地の採光・通風阻害防止
- 北側斜線：真北側の住居の日照阻害防止（特に冬季）

### ◆ 隣地斜線が低層住居系で不要な理由

1低・2低・田園住居は55条で**絶対高さ制限**（10m/12m）があるので、隣地斜線で追加制限する必要なし。

### ◆ 北側斜線の特殊性

- 真北方向のみ規制（南からの斜線は規制対象外）
- 適用地域は住居系5地域のみ（1低・2低・田園住居・1中高・2中高）
- 数値：1.25倍 + 5m（低層系）または 1.25倍 + 10m（中高層系）

### ◆ 2項：道路斜線の後退距離緩和

建築物が前面道路の境界線から**後退**している場合、後退距離だけ道路反対側境界線が外側にあるとみなして道路斜線を緩和（実質、建物を高くできる）。

### ◆ 3項：道路斜線の前面道路12m以上での緩和

前面道路の幅員が12m以上の場合、前面道路の反対側境界線からの水平距離が前面道路幅員の1.25倍以上の区域では、係数が1.5倍に緩和。

### ◆ 試験頻出

- 3類型と適用地域の組合せ
- 1低・2低・田園住居：道路斜線＋北側斜線（隣地斜線は不要）
- 商業・工業系：道路斜線＋隣地斜線（北側斜線は不要）
- 中高層住居系：3類型すべて適用

## 56条の2：日影による中高層の建築物の高さの制限【中】

出典：建築基準法56条の2（e-Gov 一次確認済み）

### ◆ 制度（日影規制）

> 別表第四（い）欄の地域内で地方公共団体の条例で指定する対象区域内にある中高層建築物は、**冬至日**の真太陽時午前8時から午後4時（北海道は午前9時〜午後3時）の間に、平均地盤面からの一定高さの水平面に、敷地境界線からの水平距離5mを超える範囲で、一定時間以上日影となる部分を生じさせてはならない。

### ◆ 要点

- 規制対象：**条例で指定された対象区域内の中高層建築物**
- 測定日：**冬至日**（最も日影が長くなる日）
- 測定時間帯：午前8時〜午後4時（道内は午前9時〜午後3時）
- 測定高さ：地盤面からの一定高さ（条例で指定）
- 測定範囲：敷地境界線から**5m超え**の範囲

### ◆ ただし書

特定行政庁が周囲の居住環境を害さないと認めて**建築審査会の同意**を得て許可した場合は不適用。

### ◆ 試験頻出

- 測定日は**冬至日**
- 範囲は**敷地境界線から5m超え**（5m以内は対象外）
- 対象地域は条例で指定された対象区域内

## 61〜67条：防火地域・準防火地域【頻出】
出典：建築基準法61〜65条＋施行令136条の2（e-Gov 一次確認済み）。

### ◆ 位置づけ
都市計画法の**地域地区**の一種（用途地域等に重ねて指定）。密集市街地の**延焼防止**が目的。**防火地域＞準防火地域**で厳しい。

### ◆ 規模に応じた構造制限（61条＋施行令136条の2）【最頻出】
61条は規模・地域に応じた技術基準を政令に委任。施行令136条の2の規模区分（逐語確認済み）：

**防火地域**
| 規模 | 構造 |
|------|------|
| 階数3以上 **または** 100㎡超 | 耐火建築物等 |
| 階数2以下 **かつ** 100㎡以下 | 準耐火建築物等 |

**準防火地域**
| 規模 | 構造 |
|------|------|
| 地階除く4階以上 **または** 1500㎡超 | 耐火建築物等 |
| 地階除く3階で1500㎡以下／2階以下で500㎡超1500㎡以下 | 準耐火建築物等 |
| 2階以下 かつ 500㎡以下 | 一定の防火措置（木造か否か）|

- 数字：**防火＝100㎡・3階**／**準防火＝1500㎡・4階・500㎡**。階数は防火地域＝地階含む／準防火地域＝「地階を除く」（条文の書き分け）。

### ◆ 屋根・外壁・看板
- **62条 屋根**：火の粉による火災発生を防ぐ技術基準に適合。
- **63条 隣地境界線に接する外壁**【頻出】：防火・準防火地域内で**外壁が耐火構造**なら、その外壁を**隣地境界線に接して**設けられる＝民法234条（50cm離す）の特則。
- **64条 看板等**：**防火地域内**の看板・広告塔等で**屋上**設置または**高さ3m超**は主要部分を**不燃材料**で造る／覆う（準防火地域に64条の規制なし）。

### ◆ 65条：内外にわたる場合【頻出】
建物が複数地域にまたがるとき**厳しい方を建物全部に適用**。
- 防火/準防火 と 指定なし区域 → 全部に防火/準防火の規定（1項）
- 防火地域 と 準防火地域 → 全部に**防火地域**の規定（2項＝厳しい方）
- ただし**防火壁で区画**されていれば外側は緩い方でよい（各項ただし書）。
- ※[[53条：建蔽率]]7項（敷地が防火地域内外で建蔽率みなし）と混同しない（65条＝構造制限／53条7項＝建蔽率加算）。

### ◆ 試験頻出
- 規模区分（防火100㎡・3階／準防火1500㎡・4階・500㎡）。63条＝耐火外壁は隣地境界に接して設置可。65条＝またがる場合は厳しい方を全部に（防火壁区画は例外）。66・67条（特定防災街区整備地区）＝低頻度スキップ。

## 建築審査会（78〜80条）

出典：建築基準法78〜80条（e-Gov 一次確認済み）

> ※**他の条文との関係**：本条は建築審査会の組織・委員規定。建築審査会の「**同意**」が要件として登場する代表例は **43条2項2号**（接道義務の例外「許可」）と **48条15項**（用途制限のただし書「許可」）。本ノート上部の43条・48条セクションでそれぞれの同意要件を確認。「許可」と「認定」の区別もそこで整理済み。

- **78条**：建基法に規定する「同意」、審査請求に対する裁決の議決等を行わせるため、**建築主事を置く市町村・都道府県**に置く
- **79条**：委員**5人以上**。法律・経済・建築・都市計画・公衆衛生・行政にすぐれた経験と知識をもつ者から、市町村長・知事が任命
- **80条**：委員の欠格条項（破産手続開始の決定で復権なし／拘禁刑以上の刑）

## 91条：建築物の敷地が地域にまたがる場合の措置【頻出】

出典：建築基準法91条（e-Gov 一次確認済み）

### ◆ 過半主義（原則）

> 敷地の過半の属する区域、地域又は地区内の建築物に関するこの法律の規定を適用する

→ 敷地の**過半（半分超）**が属する地域の規制が**敷地全体に適用**される

### ◆ 過半主義の例外（条文括弧書きで除外）

| 除外項目 | 別の処理 |
|---|---|
| **52条 容積率・53条 建蔽率** | **面積按分** |
| 54〜56条の2（高さ制限・日影規制等） | 地域ごとに適用 |
| **防火地域・準防火地域** | **厳しい方の規制が建物全体に**（別途61条等） |
| 高度地区 | 別途 |
| 22条1項市街地の区域 | 別途 |

### ◆ 規制ごとの取扱い対比【最頻出】

| 規制 | またがった場合 |
|---|---|
| **用途制限（48条）** | **過半主義**（過半地域の用途制限が敷地全体に適用） |
| **容積率・建蔽率** | 面積按分 |
| **防火地域・準防火地域** | 厳しい方が建物全体に |

### ◆ 試験ひっかけ

- 「敷地が1種低層住専と近隣商業にまたがる場合、近隣商業部分だけならホテル可」→ **誤り**（過半主義で敷地全体に過半地域の規制適用）
- 「容積率も過半主義で過半地域の率を全体に適用」→ **誤り**（容積率は面積按分）

# 農地法

出題科目（法令上の制限）。出典：農地法1〜3条・3条の2・3条の3（e-Gov 一次確認済み）。4条・5条（転用・転用目的権利移動）は Day6。

## 農地法の全体像【導入】

- **目的（1条）**：農地は食料供給の限られた資源 → **農地を効率的に利用する耕作者に権利を取得させ、耕作者の地位の安定と農業生産の増大、国民への食料の安定供給を図る**（講学上「**耕作者主義**」と呼ばれる理念。※「耕作者主義」は条文の用語ではない）
- **規制の2本柱**：
  1. **権利移動の規制**（3条＝農地のまま権利者が変わる）← 今日
  2. **転用の規制**（4条＝自分の農地を農地以外に／5条＝転用目的の権利移動）← Day6
- **3条/4条/5条 許可権者の対比【最頻出】**：3条＝**農業委員会**／4条・5条＝**都道府県知事等**

## 2条：定義（農地＝現況主義）【頻出】

> この法律で「農地」とは、**耕作の目的に供される土地**をいい、「採草放牧地」とは、農地以外の土地で、主として耕作又は養畜の事業のための採草又は家畜の放牧の目的に供されるものをいう。（2条1項・逐語）

- **現況主義**：農地かどうかは**登記簿の地目ではなく、現に耕作の目的に供されているか**で判断
  - 地目「宅地」でも現に畑として耕作 → **農地**
  - 地目「畑」でも現にアスファルト舗装の駐車場 → **農地でない**
- 補足（運用・通説／逐語未確認）：今は作付けしていない**休耕地・不耕作地でも、いつでも耕作できる状態なら農地**。舗装等で復元困難なほど確定的に転用された土地は農地でない。

## 3条：権利移動の許可【最頻出】

### ◆ 3条1項：許可の対象・許可権者

> 農地又は採草放牧地について**所有権を移転**し、又は**地上権、永小作権、質権、使用貸借による権利、賃借権若しくはその他の使用及び収益を目的とする権利を設定し、若しくは移転**する場合には、…当事者が**農業委員会の許可**を受けなければならない。ただし、次の各号…及び第五条第一項本文…は、この限りでない。

- 対象＝**農地のまま、権利者（耕作する人）が変わる**取引（転用は含まない）
- 許可権者＝**農業委員会**
- **6項：許可を受けないでした行為は、その効力を生じない（＝無効）**。許可は効力発生要件（取消しではない）

### ◆ 許可の要否と「明文／解釈」の区別【鉄則A的整理】

| 場面 | 許可 | 根拠の出典タイプ |
|------|:----:|----------------|
| 農地に**抵当権**設定 | 不要 | **解釈**（列挙権利の反対解釈＋抵当権は「使用収益目的権利」でない性質。明文の除外規定なし。質権は明文列挙だが抵当権は非列挙）|
| **単純相続**（包括承継）| 不要 | **解釈**（柱書「当事者が許可」＝法律行為前提）。届出義務は3条の3に**明文** |
| **遺産分割**等 | 不要 | **明文**（1項12号）。ただし届出は必要（12号は3条の3の免除から除外）|
| 国・都道府県が取得 | 不要 | **明文**（1項5号）|

### ◆ 3条2項：許可基準（許可してはならない場合）＝耕作者主義の具体化

「次の各号に該当する場合はすることができない」。宅建で問われる4つ：
- **1号 全部効率利用要件**：取得後に農地等の**全てを効率的に利用**して耕作等を行うと認められない
- **2号 法人制限**：**農地所有適格法人以外の法人**が取得しようとする
- **4号 常時従事要件**：取得者等が**農作業に常時従事**すると認められない
- **6号 地域調和要件**：周辺地域の農地の効率的・総合的利用に**支障を生ずるおそれ**

→ 効率的に耕作しない者・不在地主・投機目的・地域を乱す者を排除（耕作放棄・投機の防止）

### ◆ 3条3項：一般法人の賃借参入（2項2号・4号の例外）

農地所有適格法人**以外の法人**でも、**使用貸借・賃借権**なら一定要件で許可可：
- 1号：適正利用しないとき**契約を解除する旨の条件**が書面契約にある（**解除条件付き**）
- 2号：地域の他の農業者と適切な役割分担で**継続的・安定的に農業経営**
- 3号：法人の場合、役員等の**1人以上が耕作等に常時従事**

→ 一般法人は「**借りて農業＝可**／**所有＝不可**」。所有を認めると耕作放棄・投機・転用を事後排除できないが、解除条件付き賃借なら取り戻せる

### ◆ 3条の2：許可の取消し等（3項参入者への是正）

- 1項：3項許可を受けた使用貸借・賃借権者が適正利用しない等 → 農業委員会は**必要な措置を勧告できる**
- 2項：一定の場合、農業委員会は3項によりした許可を**取り消さなければならない**（義務的）
- 「解除条件付き賃借（3項）→ 不適正利用なら勧告・取消し（3条の2）」で参入容認と農地保護を両立

### ◆ 3条の3：権利取得の届出

> 3条1項本文の権利を取得した者は、**許可を受けた場合・各号（12号16号を除く）該当の場合・省令の場合を除き**、遅滞なく農業委員会に**届け出なければならない**。

- 相続など**許可不要で取得した場合も届出が必要**（「許可不要＝何もしなくてよい」ではない）

## 税理士実務との接続

- 相続による農地取得：3条許可は不要だが**届出**が必要（3条の3）。相続税の農地の納税猶予と併せて押さえる
- 特定遺贈（相続人以外）など包括承継でないものの許可要否は扱いが分かれる論点（**逐語未確認・宿題**）

## 4条・5条：転用の規制【最頻出】

### ◆ 3条・4条・5条の2軸整理

| | 3条 権利移動 | 4条 転用 | 5条 転用＋権利移動 |
|---|---|---|---|
| 場面 | 農地のまま権利者交代 | 自分で農地→非農地 | 転用目的で権利移動 |
| 2軸 | 権利移動○・転用✗ | 権利移動✗・転用○ | 権利移動○・転用○ |
| 許可権者 | **農業委員会** | **都道府県知事等** | **都道府県知事等** |
| 市街化区域特例 | **なし**（許可必要）| 農業委員会へ**届出**で許可不要 | 農業委員会へ**届出**で許可不要 |
| 無許可の効力 | **無効**（3条6項）| 51条で原状回復等の措置（私法無効はなし）| **無効**（5条3項で3条6項準用）|

### ◆ 4条 農地の転用の制限
- **自分の農地を農地以外にする**（権利移動なし）→ **都道府県知事等**（指定市町村内は市町村長）の許可
- **市街化区域内**の農地は、あらかじめ**農業委員会へ届出**で許可不要（4条1項7号）

### ◆ 5条 転用目的の権利移動の制限
- 農地を農地以外にするため**権利を設定・移転**（転用＋権利移動）→ **都道府県知事等**の許可
- **市街化区域内**は農業委員会へ**届出**で許可不要（5条1項6号）

### ◆ 無許可の効力・是正
- 3条違反・**5条違反 → 無効**（5条3項で3条6項を準用。逐語確認済）
- **4条違反**：権利移動がなく私法上の無効はない。**51条**で都道府県知事等が**許可取消・工事停止命令・原状回復等の措置**を命じられる（＋罰則）

### ◆ 市街化区域特例の急所【頻出ひっかけ】
- 4条・5条（転用が絡む）は市街化区域内なら**届出で許可不要**
- **3条（農地のまま）には特例なし**＝市街化区域内でも許可必要（農地が農地のまま残るため）

# 国土利用計画法

出題科目（法令上の制限）。出典：国土利用計画法14条1項・23条・47条（e-Gov 一次確認済み／lawdata 取得）。今日は事後届出制（23条）。勧告（24条）・24〜27条は6/13、事前届出制（注視・監視区域）・許可制（規制区域）は別途。

## 全体像：土地取引規制の3制度【導入】

| 制度 | 区域 | 中身 | 無許可・無届出の契約 |
|---|---|---|---|
| 許可制 | 規制区域（12条で指定）| 契約**前**に知事の**許可** | **無効** |
| 事前届出制 | 注視区域・監視区域 | 契約**前**に届出（価格も審査）| 有効 |
| **事後届出制** | 上記以外の一般区域 | 契約**後**に届出（利用目的をチェック）| **有効** |

→ 事後届出制は取引を止めるのでなく、取引後に利用目的を把握し不適切なら**勧告**する仕組み。だから契約は有効。

## 23条：事後届出制【最頻出】

### ◆ 何が届出対象か＝「土地売買等の契約」（14条1項）

定義：**対価を得て**行われる、**所有権・地上権その他の使用及び収益を目的とする権利**の、**移転又は設定**の契約（**予約を含む**）。対象かどうかは**3つの軸**で切る：

| 軸 | 対象外になるもの |
|---|---|
| ①契約によること | 相続・時効取得・収用・遺贈（＝契約でない）|
| ②対価を得て（有償）| 贈与・使用貸借（＝無償）|
| ③使用収益目的の権利 | 抵当権・質権など担保権（＝使用収益目的でない）|

- ○例：売買・交換・有償の地上権設定・有償の土地賃借権設定
- ✗例：贈与（②）・抵当権設定（③）・相続（①）

### ◆ 届出の要件

| 項目 | 内容 |
|---|---|
| 届出義務者 | **権利取得者（買主）のみ**（売主は不要）|
| 届出先 | 土地所在の**市町村長を経由して都道府県知事** |
| 期間 | 契約締結日から起算して**2週間以内** |
| 届出事項 | 当事者・契約日・土地の所在面積・権利の種別内容・**移転設定後の利用目的**・対価の額・省令事項 |

→ 買主だけでよい理由：制度がチェックするのは「取得後の**利用目的**が適正か」。利用目的を実現するのは取得した買主だから。価格（対価の額）も届出事項だが、**事後届出では価格は勧告対象外**（価格規制は事前届出の役割）。

### ◆ 面積要件（23条2項1号）【最頻出】

この面積**以上**で届出が必要（未満なら不要）：

| 区域 | 面積 |
|---|---|
| 市街化区域 | **2,000㎡** |
| 都市計画区域内（市街化区域以外＝市街化調整区域・非線引き）| **5,000㎡** |
| 都市計画区域外（その他）| **10,000㎡（1ha）**|

- 開発・利用への影響が大きい区域ほど基準が小さい（市街化区域が最も厳しい）。
- **一団の土地（買いの一団）【頻出】**：1号の括弧書き。個々の契約が面積未満でも、**権利取得者が一団の土地として合計面積以上**を取得するなら届出必要。面積分割による潜脱を防ぐ。判定は権利取得者基準＝「買いの一団」。

### ◆ 適用除外（23条2項2号・3号）

| 号 | 除外される場合 | 理由 |
|---|---|---|
| 2号 | 規制区域・注視区域・監視区域に所在する土地の契約 | 許可制・事前届出制の対象＝二重規制回避 |
| 3号 | 民事調停法の調停による／当事者の一方又は双方が**国等**／その他政令で定める場合 | 裁判所・公的主体の関与で適正担保 |

### ◆ 無届出の効果【最頻出】

- **契約自体は有効**（届出制であって許可制ではない＝私法上の効力に影響なし）。
- 罰則あり：**47条1項**——届出をしなかった者（1号）・虚偽の届出をした者（3号）＝いずれも**6月以下の拘禁刑又は100万円以下の罰金**。
- 対比：規制区域の許可制（14条）は無許可契約が**無効**。

## 24〜27条：勧告等（事後届出制の続き）【頻出】

出典：国土利用計画法24・25・26・27・27条の2（e-Gov 一次確認済み）。

### ◆ 24条：勧告
- 知事は、届出に係る**利用目的**が〔①土地利用基本計画等に**適合せず**②周辺の適正合理的な土地利用に**著しい支障**〕と認めるとき、**土地利用審査会の意見を聴いて**、**利用目的の変更**を勧告できる。
- **勧告の対象は「利用目的」のみ。対価の額（価格）は勧告できない**【最頻出】。
- 期間：**届出日から3週間以内**（2項）。実地調査等で**3週間の範囲で延長可**（3項・最大6週間）。

### ◆ 25〜27条・27条の2
| 条 | 内容 |
|---|---|
| 25条 | 勧告に基づき**講じた措置の報告**をさせることができる |
| 26条 | 勧告に**従わないとき**、その旨と勧告内容を**公表できる** |
| 27条 | 利用目的が変更された場合、権利処分の**あっせん等**に努める |
| 27条の2 | **助言**（届出者に有効適切な利用について助言できる）|

### ◆ 勧告に従わない場合【最頻出】
- **罰則なし・契約は有効**。制裁は**26条の公表のみ**（勧告は行政指導）。
- 対比：**届出を怠る／虚偽届出**は**47条で罰則**（6月以下の拘禁刑又は100万円以下の罰金）。「届出（義務）違反＝罰則」「勧告不服従＝公表のみ」を区別。

## 事前届出制（注視区域・監視区域）【骨格】

出典：国土利用計画法27条の3〜27条の7（e-Gov 一次確認済み）。事後届出（23条）との対比で骨格のみ。深掘りはR2。

| 項目 | 事後届出（23条）| 事前届出・注視区域（27条の4）|
|---|---|---|
| 区域 | 一般区域 | 注視区域（27条の3指定＝地価が相当程度超えて上昇のおそれ）|
| 時期 | 契約**後**2週間以内 | 契約**前**（あらかじめ）|
| 届出者 | **権利取得者（買主）のみ** | **両当事者** |
| 予定対価の額（価格）| 届出も審査もなし | **届出する・審査される** |
| 待機期間 | なし | **6週間**は契約締結禁止 |
| 勧告対象（27条の5）| 利用目的のみ | **予定対価の額＋利用目的** |

- **監視区域（27条の6指定・27条の7届出）**：地価が**急激に**上昇のおそれ。27条の4を準用しつつ、**面積要件を都道府県の規則で引き下げる**（より厳しい）。
- 3制度：許可制（規制区域・無許可は契約**無効**）／事前届出（注視・監視・契約前・両当事者・価格審査・6週間待機）／事後届出（一般区域・契約後・買主のみ・利用目的のみ）。

## 国土利用計画法 次回・宿題
- 24〜27条（勧告・報告・公表・あっせん）・27条の2（助言）＝**6/13完了**／事前届出制（注視・監視区域27条の3〜7）の骨格も6/13完了

- 届出事項の省令細目・「国等」「政令で定める場合」の具体的範囲（施行令）→ R3直前

# 盛土規制法（宅地造成及び特定盛土等規制法）

出典：宅地造成及び特定盛土等規制法（lawid 336AC0000000191）／同施行令（lawid 337CO0000000016）（e-Gov 逐語確認済み）

## 全体像【導入】
- 元は**宅地造成等規制法**（昭和36年）。市街地の**宅地造成**が主な規制対象だった。
- **2021年の熱海・伊豆山の土石流災害**（不適正盛土の崩落）を受け、2022年に大改正・改称→**「盛土規制法」**。狙いは**宅地造成に限らず森林・農地を含めどこでも危険な盛土を規制**すること。
- 規制対象＝**「宅地造成等」**（3本柱）：
  - **宅地造成**＝宅地**以外**を**宅地にするため**の盛土等（政令の規模）
  - **特定盛土等**＝宅地・農地等の盛土等で、隣接・近接する宅地に**災害を発生させるおそれが大きいもの**
  - **土石の堆積**＝宅地・農地等の土石の堆積（一定期間後に除却するもの）

## 規制区域の2種類（指定権者はいずれも都道府県知事）【最頻出】
| 区域 | 条文 | 指定要件 | 想定エリア |
|---|---|---|---|
| **宅地造成等工事規制区域** | 10条 | **市街地・市街地となろうとする土地・集落**で災害のおそれが大きい | 人が住むエリア |
| **特定盛土等規制区域** | 26条 | 傾斜度・渓流等の**自然的条件**から危害のおそれが**特に大きい** | 市街地外（山間部等）も含む |

## 許可制（12条・30条）【最頻出】
- いずれも**都道府県知事の許可**・**工事着手前**（事前許可）。

| | 宅地造成等工事規制区域(12条) | 特定盛土等規制区域(30条) |
|---|---|---|
| 対象 | **宅地造成等**（3本柱すべて）| **特定盛土等・土石の堆積**のみ（宅地造成は対象外）|
| 規模 | 政令規模（施行令3条・4条）| **大規模**（政令で定める規模）に限る |

## 許可が必要な工事の規模（施行令3条・4条）【頻出・数値】
**施行令3条（宅地造成・特定盛土等）：**
| 号 | 工事 | 規模 |
|---|---|---|
| 1号 | 盛土で崖を生じる | 高さ**1m超**の崖 |
| 2号 | 切土で崖を生じる | 高さ**2m超**の崖 |
| 3号 | 盛土＋切土を同時に行い崖を生じる | 高さ**2m超**の崖 |
| 4号 | 1・3号以外の盛土 | 高さ**2m超** |
| 5号 | 上記以外の盛土・切土 | 面積**500㎡超** |

**施行令4条（土石の堆積）：** 1号 高さ**2m超**／2号 面積**500㎡超**

→ **盛土は1m超・切土は2m超**から規制。盛土は人工的に積んだ土で締固め不足だと崩れやすく不安定（切土は安定した自然地盤を削る）ため、盛土の基準が厳しい。熱海も盛土の崩落。

## 届出制（21条・27条・40条）【頻出】
| 場面 | 期限 | 条文 |
|---|---|---|
| 規制区域の**指定時に既に行われている**工事 | 指定日から**21日以内**（事後）| 21条1項（特定盛土等規制区域は40条）|
| **特定盛土等規制区域**で許可規模に**満たない**特定盛土等・土石の堆積 | 着手の**30日前**まで（事前）| **27条1項** |
| 擁壁・排水施設等の工事（政令）| 着手の**14日前**まで | 21条3項 |
| 公共施設用地を宅地・農地等に**転用** | 転用日から**14日以内** | 21条4項 |

- 特徴：特定盛土等規制区域は、許可不要の規模でも**着手30日前までの事前届出**が必要（27条）。

## 許可不要の調整＝開発許可みなし（15条2項・34条2項）【頻出ひっかけ】
- 規制区域の指定後に**都市計画法29条の開発許可**を受けたとき → 盛土規制法の許可（12条・30条）を**受けたものとみなす**。二重規制の回避。
- 「開発許可と盛土規制法の許可の**両方が必要**」は誤り。

## 宿題（R2送り・低頻度）
- 完了検査・中間検査・定期報告／監督処分・改善命令・保全義務／罰則 → R2過去問・直前期で拾う。
- 施行令の「災害のおそれがない政令工事」（27条但書）の具体的範囲 → R3直前。

# 土地区画整理法

出典：土地区画整理法76・89・96・98・99・100・103・104条（e-Gov逐語確認済み・lawid=329AC0000000119）。

## 全体像【導入】
- 不整形・道路未整備の区域の区画を整え、道路・公園等の公共施設を整備する事業。中心手法＝**換地**（従前の宅地を整理後の新しい宅地に置き換える）。
- **減歩**＝地権者が土地を出し合い宅地面積が減る。内訳＝公共施設用地の**公共減歩**＋売却用の**保留地減歩**。減歩しても整備で**単価が上がり**価値の総額は保たれる（→89条照応原則）。
- 流れ：事業計画→（工事中）**仮換地指定**→**換地処分**で最終確定。

## 仮換地（98・99・100条）【最頻出】
- **98条 指定**：施行者は換地処分前に仮換地を指定できる。
- **99条 効果**：効力発生日〜換地処分公告日まで、**仮換地**を従前と同じ**使用・収益**でき、**従前の宅地は使用収益できない**。
- **所有権は移らない**（条文は「使用・収益」のみ）。所有権・登記名義は従前地のまま＝**頻出ひっかけ**（「仮換地指定で所有権取得」は誤り）。
- **100条**：施行者は必要なとき従前地の使用収益を停止できる。

## 換地処分（103・104条）【最頻出】
- **103条 手続**：関係権利者に通知（1項）。原則は**工事完了後**遅滞なく（2項・ただし書で例外）。**公告**（4項）。
- **104条 効果（すべて公告の翌日）**：①1項＝**換地は従前の宅地とみなされる**（所有権が確定的に換地へ）・換地を定めなかった従前地の権利は公告日終了時に消滅／②11項＝**保留地は施行者が取得**／③8項＝**清算金が確定**（過不足は94条の清算金で金銭精算）。

## 換地照応の原則（89条）【頻出】
- 換地は従前の宅地と**位置・地積・土質・水利・利用状況・環境等が照応**するように定める。従前地に見合った換地を保障し地権者の財産権を保護。

## 建築行為等の制限（76条）【頻出】
- 各号公告日後〜換地処分公告日まで、施行地区内で**事業の施行の障害となるおそれ**がある①土地の形質変更②建築物等の新築改築増築③政令で定める移動困難物件の設置堆積は**許可制**。
- 許可権者＝国土交通大臣施行は**国土交通大臣**、それ以外は**都道府県知事等**（市の区域内で個人施行者・組合・区画整理会社施行、又は市が3条4項施行の場合は**市長**）。
- 趣旨＝事業の円滑な施行（工事妨害・移転補償増大・換地設計支障の防止）。

## 仮換地 vs 換地処分 まとめ
| 局面 | 移るもの | 従前地 | 根拠 |
|---|---|---|---|
| 仮換地指定（工事中） | 使用収益権能のみ（**所有権は移らない**） | 使用収益できない | 98・99 |
| 換地処分（公告の翌日） | **所有権**（換地＝従前地とみなす）／保留地は施行者取得／清算金確定 | 権利消滅 | 103・104 |
