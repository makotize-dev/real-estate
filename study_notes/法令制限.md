## 都市計画法の全体像

### ◆ 目的（1条）

- 都市の健全な発展と秩序ある整備を図り、国土の均衡ある発展と公共の福祉の増進に寄与する
- 手段：「計画」を立てて、それに沿って土地利用を規制する

### ◆ 規制の入口は「区域指定」

日本中すべての土地が同じように規制されるわけではない。3区分：

| 区分 | 内容 | 規制の強さ |
|---|---|---|
| **都市計画区域** | 一体の都市として総合的に整備・開発・保全する必要がある区域 | 強 |
| **準都市計画区域** | 都市計画区域**外**で、相当数の建築物等の建築・造成が現に行われ／見込まれる区域 | 中（限定的） |
| それ以外 | 上記いずれにも指定されない区域 | 都市計画法の規制対象外 |

## 4条：用語の定義（開発行為・特定工作物）

### ◆ 4条12項：開発行為の定義

> 「主として**建築物の建築**又は**特定工作物の建設**の用に供する目的で行なう**土地の区画形質の変更**」

→ 2要素で構成：
1. 主として建築物の建築 or 特定工作物の建設の用に供する目的
2. 土地の区画形質の変更を伴う

### ◆ 4条11項：特定工作物の定義

| 区分 | 内容 | 例 |
|---|---|---|
| **第一種特定工作物** | 周辺環境を**悪化させるおそれがある工作物**（政令で指定） | コンクリートプラント、アスファルトプラント等 |
| **第二種特定工作物** | **大規模な工作物**（政令で指定） | ゴルフコース、運動・レジャー施設、墓園等 |

### ◆ 「区画形質の変更」の3要素（行政解釈）

**出典**：国土交通省「**開発許可制度運用指針**」（平成26年8月1日 国都計第67号、最終改正 令和7年3月19日 国都計第193号）。地方自治法245条の4に基づく技術的助言

| 要素 | 意味 |
|---|---|
| **区画** | 敷地を分割・統合して形を変えること |
| **形** | 切土・盛土などで高さを変えること |
| **質** | 用途を変更すること（例：農地→宅地） |

→ 3要素のいずれかに該当すれば「区画形質の変更」あり

## 5条：都市計画区域の指定

### ◆ 指定権者【頻出】

| 場面 | 指定権者 |
|---|---|
| 原則 | **都道府県** |
| 2以上の都府県にまたがる場合 | **国土交通大臣**（関係都府県の意見聴取） |

→ ひっかけ：「2以上にまたがる場合は知事の協議で決める」は誤り

### ◆ 指定の対象

- 市又は政令で定める要件に該当する町村の中心の市街地を含む区域
- 自然的・社会的条件、人口・土地利用・交通量等を勘案して、一体の都市として総合的に整備等する必要がある区域

---

## 5条の2：準都市計画区域

### ◆ 指定権者・対象

- 指定権者：**都道府県**
- 対象：都市計画区域**外**で、相当数の建築物等の建築・造成が現に行われ／見込まれる区域、放置すれば将来の都市整備に支障が生じるおそれがある区域

### ◆ 趣旨

都市計画区域外でも、開発が進みつつある場所を放置すると将来困る → 先回りして規制をかける

## 7条：区域区分（線引き）

### ◆ 区域区分の任意/義務【頻出】

| 場面 | 扱い | 根拠 |
|---|---|---|
| 原則 | 区域区分を定めることが**できる**（任意） | 7条1項本文 |
| 三大都市圏の特定地域・大都市に係る都市計画区域として政令で定めるもの（政令指定都市等） | 区域区分を定める**ものとする**（義務） | 7条1項但書 |

→ ひっかけ：「すべての都市計画区域で区域区分が必要」は誤り

### ◆ 義務となる三大都市圏（7条1項1号）

- イ：首都圏整備法に基づく既成市街地・近郊整備地帯
- ロ：近畿圏整備法に基づく既成都市区域・近郊整備区域
- ハ：中部圏開発整備法に基づく都市整備区域

### ◆ 市街化区域・市街化調整区域の定義

| 区分 | 定義 |
|---|---|
| **市街化区域**（7条2項） | 既に市街地を形成している区域 + おおむね**10年以内**に優先的かつ計画的に市街化を図るべき区域 |
| **市街化調整区域**（7条3項） | 市街化を**抑制**すべき区域 |

### ◆ 「区域区分が定められていない都市計画区域」（俗称：未線引き・非線引き）

- 法律上の用語は「区域区分が定められていない都市計画区域」（条文に「未線引き」「非線引き」の語はない）
- 規制対象外ではない：用途地域・開発許可等の規制はかかりうる
- 「規制対象外」になるのは都市計画区域・準都市計画区域いずれにも指定されていない区域

## 8〜9条：地域地区・用途地域

### ◆ 用途地域とは

土地の利用方法を制限する地域地区。**建ててもいい建物・建ててはいけない建物**を決める

### ◆ 用途地域13種類（9条）【頻出】

| グループ | 種類数 | 用途地域 |
|---|---:|---|
| **住居系** | 8 | 1.第一種低層住居専用 / 2.第二種低層住居専用 / 3.第一種中高層住居専用 / 4.第二種中高層住居専用 / 5.第一種住居 / 6.第二種住居 / 7.準住居 / 8.田園住居 |
| **商業系** | 2 | 9.近隣商業 / 10.商業 |
| **工業系** | 3 | 11.準工業 / 12.工業 / 13.工業専用 |

### ◆ 性格の方向性

- 住居系（特に「専用」がつく地域）：**住環境保護が最も厳しい**、商業・工業施設は厳しく制限
- 商業系：商業・業務の利便を優先、住居も可能
- 工業系（特に「工業専用」）：**工業利便を優先**、工業専用地域は**住宅も建てられない**

→ 住居系 → 商業系 → 工業系 の順に、住環境保護が弱まる

### ◆ 住居系8種類のサブグループ

| サブグループ | 含まれる用途地域 | 特徴 |
|---|---|---|
| 低層住居専用 | 第一種・第二種 | 一戸建て中心、3階建て程度まで |
| 中高層住居専用 | 第一種・第二種 | マンション可、用途制限は厳しめ |
| 住居 | 第一種・第二種・準住居 | 「専用」がつかない、店舗・事務所もある程度可 |
| 田園住居 | 田園住居【2018年新設】 | 農業利便と住居環境の調和 |

→ 住居系内：**第一種＝厳しい／第二種＝緩い／準＝最も緩い**

## 13条1項7号：用途地域の定め方【頻出】

| 区域 | 用途地域の扱い | 条文文言 |
|---|---|---|
| **市街化区域** | **必ず定める** | 「少なくとも用途地域を定めるものとし」 |
| **市街化調整区域** | **原則として定めない**（例外あり） | 「原則として用途地域を定めないものとする」 |
| 区域区分が定められていない都市計画区域 | 13条1項7号には明示規定なし | — |
| 準都市計画区域 | 別規定（13条3項） | — |

### ◆ 論理

- 市街化区域：市街化を進める → 用途整理が必要
- 市街化調整区域：市街化を抑制する → 積極的な土地利用を想定しない → 用途地域は原則不要

## 29条：開発許可【頻出】

### ◆ 制度の趣旨

無秩序な市街化を防止し、計画的な市街化を誘導するため、一定の開発行為に**事前許可**を必要とする制度

### ◆ 許可権者

**都道府県知事**（指定都市・中核市等の区域内では当該指定都市等の長）

### ◆ 許可対象（29条1項：都市計画区域・準都市計画区域内）

開発行為（4条12項）をしようとする者は、原則として許可必要。ただし規模要件あり。

### ◆ 規模要件【頻出】（施行令19条）

| 区域 | 許可が必要な開発行為の規模 |
|---|---|
| **市街化区域** | **1,000㎡以上** |
| **市街化調整区域** | **規模を問わず許可必要**（面積要件なし） |
| **区域区分が定められていない都市計画区域**（非線引き） | **3,000㎡以上** |
| **準都市計画区域** | **3,000㎡以上** |
| **都市計画区域・準都市計画区域外** | **10,000㎡（1ha）以上** |

→ 市街化調整区域の**面積要件なし**が最重要。市街化抑制区域なので小規模開発も規制

### ◆ 条例による引下げ

都道府県・指定都市等の条例で、開発許可を要する規模を**300㎡まで**引き下げ可能

### ◆ 「開発行為」に該当する典型例

| 例 | なぜ該当するか |
|---|---|
| 農地を切土・盛土して住宅地を造成し戸建て分譲 | 区画形質変更＋建築物の建築目的 |
| 山林を伐採・整地してマンション用地にする | 形質変更＋建築物の建築目的 |
| 既存宅地を分筆・道路新設して建売分譲 | 区画変更＋建築物の建築目的 |
| 山林を切り開いてゴルフコース造成 | 形質変更＋第二種特定工作物の建設目的 |
| コンクリートプラント設置のため造成 | 形質変更＋第一種特定工作物の建設目的 |

### ◆ 「開発行為」に該当しない例【ひっかけ】

| 例 | なぜ該当しないか |
|---|---|
| 既存宅地に新築するだけ（造成等を伴わない） | 区画形質変更がない |
| 農地を青空駐車場にする（建物なし） | 建築物・特定工作物の建設目的でない |
| 山林を伐採して資材置場にする（建物なし） | 建築物・特定工作物の建設目的でない |
| 単なる分筆（土地登記上の分割のみ） | 物理的な区画形質変更がない |

→ 出典：宅建参考書・解説サイトの典型的説明（行政解釈の正確な原典は開発許可制度運用指針を要確認）

### ◆ 判定フロー

```
1. 当該行為は「開発行為」か？
   - 主として建築物の建築 or 特定工作物の建設の用に供する目的か
   - 土地の区画形質の変更を伴うか
   → Yes なら 開発行為

2. 開発行為の場所・規模は？
   - 市街化区域：1,000㎡以上 → 許可必要
   - 市街化調整区域：規模問わず → 許可必要
   - 非線引き・準都市計画区域：3,000㎡以上 → 許可必要
   - 都市計画区域外：10,000㎡以上 → 許可必要

3. 許可不要のケース（29条1項各号）に該当するか？
   - 詳細は下記「29条1項各号」セクション参照
```

## 29条1項各号：許可不要の例外（11号）【頻出】

### ◆ 11号の分類と立法趣旨

| 号 | 内容 | 立法趣旨カテゴリ |
|---|---|---|
| **1号** | 規模未満（市街化区域・非線引・準都計のみ） | 規模軽微 |
| **2号** | 農林漁業用建築物（市街化調整区域・非線引・準都計のみ） | 地域産業の保護 |
| **3号** | 駅舎・図書館・公民館・変電所等の**広域**公益建築物 | 公益施設 |
| **4号** | 都市計画事業の施行 | 他法令で代替審査済み |
| **5号** | 土地区画整理事業の施行 | 同 |
| **6号** | 市街地再開発事業の施行 | 同 |
| **7号** | 住宅街区整備事業の施行 | 同 |
| **8号** | 防災街区整備事業の施行 | 同 |
| **9号** | 公有水面埋立で告示前 | 別法（公水法）で管理中 |
| **10号** | 非常災害の応急措置 | 緊急性 |
| **11号** | 通常の管理行為・軽易な行為 | 規模軽微・規制になじまない |

### ◆ 区域別の例外適用範囲【頻出】

| 区域 | 「規模軽微」例外（1号） | 「農林漁業」例外（2号） | 3〜11号 |
|---|:---:|:---:|:---:|
| 市街化区域 | ✅ 使える | ❌ 使えない | ✅ 使える |
| 市街化調整区域 | ❌ **使えない** | ✅ 使える | ✅ 使える |
| 非線引・準都計 | ✅ 使える | ✅ 使える | ✅ 使える |

→ 市街化調整区域では「**規模が小さくても許可必要**」というのが重要。
→ 「市街化区域は規模で逃げる／市街化調整区域は農林漁業で逃げる」で覚える。

### ◆ 4〜9号の市街化調整区域での実質的適用可能性

条文上は4〜9号いずれも区域限定なしだが、各事業の制度的性質から：

| 号 | 実質的可能性 | 補足 |
|---|---|---|
| 4号 都市計画事業 | **ありうる** | 市街化調整区域を通過する都市計画道路など |
| 5号 土地区画整理事業 | ありうる（実例少） | 「市街化抑制」と矛盾しうるので慎重判断 |
| 6号 市街地再開発事業 | 事実上なし | 「市街地」が前提 |
| 7号 住宅街区整備事業 | 事実上なし | 大都市住宅供給制度 |
| 8号 防災街区整備事業 | 事実上なし | 密集市街地が前提 |
| 9号 公有水面埋立 | ほぼなし | 市街化調整区域指定の公水面は内陸湖沼の極稀例外 |

→ 試験では**条文上の区域限定の有無で機械的に判断**する（実質可能性まで問われることは稀）

### ◆ 試験ひっかけパターン

- 「市街化調整区域で小規模な住宅開発 → 規模未満で許可不要」→ **誤り**（1号は市街化調整区域で使えない）
- 「市街化区域で農家の自宅建築 → 農林漁業用で許可不要」→ **誤り**（2号は市街化区域で使えない）
- 「市街化調整区域で駅舎建築 → 開発許可必要」→ **誤り**（29条1項3号で許可不要、区域限定なし）

## 33条：開発許可の基準（技術基準）【頻出】

### ◆ 羈束行為であること【最重要】

33条1項本文：「都道府県知事は…許可を**しなければならない**」

→ **裁量ではない**。基準を満たせば必ず許可、満たさなければ必ず不許可。

### ◆ 試験のひっかけ

- 「環境上好ましくないので許可しない」「住民が反対しているので不許可」→ **誤り**（裁量判断できない）
- 「基準を満たしても許可しないことができる」→ **誤り**（羈束行為）

### ◆ 14号のグループ分け

| グループ | 号 | 何を見るか |
|---|---|---|
| **A 計画適合** | 1号・5号 | 用途地域・地区計画への適合 |
| **B 公共施設・インフラ** | 2号・3号・4号・6号 | 道路・公園・排水・給水・学校等 |
| **C 安全・環境** | 7号・8号・9号・10号 | 擁壁・排水安全／災害危険区域除外／樹木・表土保全／緩衝帯 |
| **D 利便性** | 11号 | 輸送便利性 |
| **E 申請者の適格性** | 12号・13号・14号 | 資力・信用／工事能力／関係権利者の同意 |

### ◆ 自己居住用住宅の特例

33条1項では **12〜14号の基準**（資力・信用／工事能力／関係権利者の同意）について、「**政令で定める場合に限り適用する**」となっており、自己居住用住宅（マイホーム）には**適用されない**運用。

→ 試験：「マイホームの開発許可で資力・信用の証明を求められる」→ **誤り**

※ 正確な対象範囲は施行令で規定（要確認事項）

### ◆ 条例による技術基準の強化

地方公共団体は条例で技術的細目を**強化・付加**できる。地域実情に応じた上乗せ規制が可能。

出典：e-Gov 都市計画法33条（一次確認済み）

## 34条：市街化調整区域における特別基準【頻出】

### ◆ 33条との関係【最重要】

市街化調整区域での開発許可は：

> **33条の技術基準（共通）をクリア** ＋ **34条のいずれかの号に該当**

→ **両方** が必要。市街化区域より審査が厳しい二段階構造。

### ◆ 14号 + 8の2号の論点グループ

| グループ | 号 | 立法趣旨 |
|---|---|---|
| **A 立地必然性** | 2号（鉱物・観光・第一種特定工作物）・4号（農林漁業＋加工処理） | その場所でしかできない |
| **B 公益・地域必要性** | 1号（公益建築物・日常店舗）・3号（特殊条件事業）・5号（特定農山村活性化）・6号（中小企業連携）・7号（既存工場関連）・8号（危険物施設） | 公益上の必要 |
| **C 既存権利の保護** | 8の2号（災害区域からの代替）・11号（市街化区域隣接の連たん地区）・13号（区域区分決定時の既権利者） | 既得権・救済 |
| **D 条例・計画・審査会** | 10号（地区計画適合）・12号（条例限定）・**14号**（開発審査会の議） | 個別判断の余地 |
| **E その他** | 9号（その他政令で定める） | 補充 |

### ◆ 34条14号【最頻出】

> 「**開発審査会の議を経て**、**開発区域の周辺における市街化を促進するおそれがなく**、かつ、**市街化区域内において行うことが困難又は著しく不適当**と認める開発行為」

**3要件すべて**が必要：

| 要件 | 内容 |
|---|---|
| ① 手続要件 | 開発審査会の議を経ること |
| ② 実体要件1 | 周辺の市街化促進のおそれがない |
| ③ 実体要件2 | 市街化区域でやることが困難または著しく不適当 |

→ 「審査会の議を経るだけ」で許可されるわけではない（実体要件②③も必要）

### ◆ 34条14号の機能

- 1〜13号：列挙された定型ケース
- **14号**：列挙からこぼれた非定型ケースを、審査会の慎重審議＋実体要件で**例外的に許可する道**
- 「許可しにくくする」のではなく「**1〜13号以外でも個別判断で許可する余地を与える**」規定。ただし安易に許可されないよう手続要件と実体要件で歯止め

出典：e-Gov 都市計画法34条（一次確認済み）

## 29条1項 vs 34条 混同整理【頻出】

| 同じテーマ | 29条1項（許可**不要**） | 34条（市街化調整区域：許可は**必要**だが認められる） | 区別の決め手 |
|---|---|---|---|
| **公益建築物** | **1項3号**：駅舎・図書館・公民館・変電所など**広域公益施設** | **1号**：周辺住民の**日常生活**に必要な店舗・診療所等の**地域密着型施設** | 広域 vs 地域密着 |
| **農林漁業用** | **1項2号**：施行令で定める**限定的なもの**（畜舎・農作業舎・農業者の自宅等） | **4号**：1項2号の政令で定めるもの**以外**＋**加工処理施設** | 典型 vs 加工処理 |

### ◆ 整理の核心

> **29条1項に該当する**：許可手続そのものが不要
> **29条1項に該当しない**：33条＋（市街化調整区域なら）34条で審査

→ 「許可不要」と「許可されやすい」は**別の話**

### ◆ 試験ひっかけパターン

| 問 | 結論 | 根拠 |
|---|---|---|
| 市街化調整区域で農産物加工施設は29条1項2号で許可不要 | **誤り** | 加工施設は1項2号の政令対象外。34条4号で許可必要 |
| 市街化調整区域で駅舎建築は許可必要 | **誤り** | 29条1項3号で許可不要（区域限定なし） |
| 開発審査会の議だけで34条1〜13号非該当でも必ず許可 | **誤り** | 14号は3要件構造（手続＋実体2要件） |
