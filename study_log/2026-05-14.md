### 学んだポイント

**1. 17条（合格の取消し等）— 登録ではなかった**

事前認識では「17条以下＝登録手続」と誤認していたが、実物は **試験不正対策の規定**：
- 1項：不正受験者の合格決定の取消し or 受験禁止
- 2項：指定試験機関も同様の職権
- 3項：3年以内の受験禁止

→ 17条は16条（試験）の付随規定で、登録の本体は18条以下から。

**2. 18条（宅地建物取引士の登録）**

**本文の3要件**：
- 試験合格
- 国土交通省令で定める期間以上の実務経験 or 国交大臣が同等以上と認めたもの
- **試験を行った都道府県知事**に登録申請

→ 実務経験要件（省令で2年以上）を満たさない場合は**登録実務講習**修了で代替。

**18条1項各号 12個の欠格事由**：5条（業者欠格事由）と比較で整理

| カテゴリ | 5条 | 18条 |
|---|:---:|:---:|
| 判断能力・破産・心身故障 | ✅ | ✅ |
| 刑事罰（拘禁刑・罰金刑） | ✅ | ✅ |
| 暴力団 | ✅ | ✅ |
| 行政処分歴 | ✅ | ✅ |
| **法人対応**（役員・政令使用人） | ✅ | ❌（自然人のみ） |
| **暴力団等の支配** | ✅ | ❌ |
| **事務所への専任宅建士義務** | ✅ | ❌ |
| **登録消除・事務禁止処分歴**（取引士特有） | ❌ | ✅ |
| **「取引士の事務」適正実行困難**（限定的） | ❌（一般的） | ✅ |
| **営業未許可未成年者** | （5条1項11号別構造） | ✅（1号） |

**18条1項1号の特殊性**：「営業に関し成年者と同一の行為能力を有しない未成年者」
- 民法6条（営業を許された未成年者）を引用
- 営業の許可を受けた未成年者なら登録可能（単純な年齢規制ではない）

**18条2項**：登録の方式
- 都道府県知事が**宅地建物取引士資格登録簿**に氏名・生年月日・住所・登録番号・登録年月日等を登載

**3. 19条（登録の手続）**

- 1項：登録できる者は**登録申請書**を都道府県知事に提出
- 2項：知事は申請書を受けたら**遅滞なく、登録をしなければならない**（**羈束行為**）

→ 33条（開発許可）と同じ羈束行為。要件を満たせば必ず登録、満たさなければ拒否。

**4. 19条の2（登録の移転）**

| 項目 | 内容 |
|---|---|
| 主体 | 18条1項の登録を受けている者 |
| 要件 | 登録知事の都道府県**以外**の業者事務所で業務に従事する／しようとするとき |
| 手続 | 従事先の都道府県知事に **現登録知事を経由して** 申請 |
| 法的性質 | **任意**（することができる） |
| 適用除外 | 68条2項・4項の事務禁止処分中で期間満了前 |

**5. 20条（変更の登録）**

- 18条1項の登録を受けている者は、登録事項に変更があれば**遅滞なく**変更登録を**申請しなければならない**
- **義務**（しなければならない）

**6. 19条の2 vs 20条 の決定的な違い【頻出】**

| 場面 | 条文 | 何が変わるか | 義務／任意 |
|---|---|---|---|
| 住所・氏名・本籍などの変更 | **20条**（変更の登録） | **登録事項** | **義務** |
| 勤務先事務所が他県に | **19条の2**（登録の移転） | **登録知事** | **任意** |

### 試験ひっかけパターン（演習済み）

| ケース | 結論 |
|---|---|
| 18歳・営業の許可なしの未成年者が試験合格 → 登録可能 | ❌ 不可（18条1号） |
| 17歳・営業を許可された未成年者が試験合格 → 登録可能 | ✅ 可（18条1号反対解釈） |
| 法人が宅建士登録 | ❌ 不可（取引士は自然人のみ） |
| 申請者が要件を満たすのに知事が裁量で登録拒否 | ❌ 違法（19条2項は羈束） |
| 東京都試験合格・神奈川県在住 → 神奈川県知事に登録申請 | ❌ 誤り（試験を行った都道府県知事＝東京都） |
| 取引士が住所を東京→神奈川に変更 → 19条の2で登録の移転 | ❌ 誤り（住所変更は20条） |
| 東京都登録の取引士が神奈川県業者事務所で従事 → 必ず登録の移転 | ❌ 誤り（19条の2は任意） |
| 登録移転の申請は新知事に直接 | ❌ 誤り（現登録知事を経由） |
| 取引士が住所変更 → 業者が変更申請 | ❌ 誤り（20条は本人） |

### 税理士知識との比較・つながり

- 税理士登録（税理士法18条以下）も「試験合格→登録→税理士証票」の構造で、宅建士の3段階構造と類似
- 登録の移転（19条の2）は、税理士の場合「事務所変更」が税理士会への登録変更を伴う点と類似
- 「経由して申請」する仕組みも、税理士会経由の届出手続と発想が同じ

### Claude側の反省

**① 17条が「登録」関連だと誤認していた**

5/11セッションで W2計画に「14-15条 試験・業務原則」と書いて訂正した経緯と同じく、条文番号と内容の対応を一次情報なしに想定していた。17条は実は試験不正対策で、登録の本体は18条以下だった。

**② 19条の2（登録の移転）を飛ばした**

論理順を踏まず「住所変更」の問いから20条に飛んだうえ、その手前にある19条の2（登録の移転）を素通りした。ユーザーから「19条の2もありますよ」と指摘されて気づいた。

**③ 再発防止ルール**

新しい条文範囲に入るときは、まず**該当章の条文目次を確認**してから入る。e-Gov API は条番号指定で個別取得するため、**枝番号（条の2、条の3）の存在に気づくには目次取得が必要**。

```

条番号と枝番号の有無を把握してから個別条文の取得に入る
```
