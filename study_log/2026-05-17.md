## 権利関係（民法20条・21条）

### 20条（制限行為能力者の相手方の催告権）

催告の相手方によって「確答がないとき」の効果が変わる：

| 項 | 催告の相手方 | 確答なしの効果 |
|---|---|---|
| 1項 | 行為能力者となった本人 | **追認**とみなす |
| 2項 | 法定代理人・保佐人・補助人（権限内の行為） | **追認**とみなす |
| 3項 | （1項2項の相手方）特別の方式を要する行為で方式具備の通知なし | **取消**とみなす |
| 4項 | 被保佐人・被補助人 本人（保佐人等の追認を得るべき旨を催告） | **取消**とみなす |

**核心ロジック**：催告の相手方が「**単独で追認できる人**」（行為能力者となった本人・法定代理人等）なら沈黙＝追認みなし。「**単独では追認できない人**」（被保佐人・被補助人本人）なら沈黙＝取消みなし。

**催告できる相手・できない相手**：
- 未成年者・成年被後見人 本人 → 催告できない（法定代理人へ）
- 被保佐人・被補助人 本人 → 催告できる（4項、効果は取消みなし）

催告期間は **1か月以上**。

### 21条（制限行為能力者の詐術）

- 制限行為能力者が「行為能力者である」と信じさせるため**詐術**を用いた → **取消できない**
- **単なる黙秘**（制限行為能力者であることを黙っていただけ）→ 原則 詐術にあたらない
- 黙秘が他の言動と相まって相手方を誤信させた／誤信を強めた → 詐術にあたる（判例の確立した解釈）

### 取消可能 vs 浮動的無効

- 制限行為能力者の行為：**取消可能**（一応有効、取消で遡及的無効）
- 無権代理（5/16）：**浮動的無効**（効力未発生、追認で発生）
- 共通点：どちらも不安定 → 相手方に催告権

## 法令制限（建築基準法 導入）

### 建築基準法の2本柱

| | 単体規定 | 集団規定 |
|---|---|---|
| 規律対象 | 建物そのものの安全・衛生 | 建物と周辺環境・街との関係 |
| 具体例 | 構造耐力・防火・避難・採光・換気 | 用途制限・建蔽率・容積率・高さ制限・接道義務 |
| 適用範囲 | **全国どこでも** | **都市計画区域・準都市計画区域内** |
| 趣旨 | その建物にいる人の安全 | 良好な市街地環境の形成 |

→ 集団規定が「都市計画区域・準都市計画区域内」だけなのは、「街として整備する区域でだけ意味がある」から。都市計画法の区域指定と建築基準法の集団規定の適用範囲がリンク。都計法と建基法はセットで機能する。

## 税（登録免許税 導入）

### 課税標準の3類型

| 登記の種類 | 課税標準 |
|---|---|
| 所有権の保存・移転登記 | 不動産の価額（固定資産課税台帳の登録価格） |
| 抵当権の設定登記 | **債権金額**（又は極度額） |
| 一部の登記 | 不動産の個数 |

### 本則税率（出典：国税庁タックスアンサー No.7191）

| 登記の種類 | 本則税率 |
|---|---|
| 所有権保存登記 | 1,000分の4（0.4%） |
| **売買**による所有権移転登記 | **1,000分の20（2.0%）** |
| **相続**による所有権移転登記 | **1,000分の4（0.4%）** |
| 贈与による所有権移転登記 | 1,000分の20（2.0%） |
| 抵当権設定登記 | 1,000分の4（0.4%・課税標準は債権金額） |

### 納税義務者（登録免許税法3条、e-Gov確認済み）

- 登記等を受ける者が納税義務
- 登記等を受ける者が**2人以上**あるとき → **連帯**して納付義務
- 売買による所有権移転登記は売主・買主が共同申請 → **売主・買主が連帯納税義務**（買主のみではない）

### 試験頻出ポイント

- 売買（2.0%）と相続（0.4%）で所有権移転登記の税率が違う
- 抵当権設定登記の課税標準は債権金額（不動産の価額ではない）
- 納税義務者は連帯
- 住宅用家屋の軽減措置あり（租税特別措置法。住宅取得資金借入の抵当権設定は1,000分の1＝措法75条）
