# past_exams — 実物の年度別過去問（ローカル保存専用）

R2以降の週末に解く**実物の宅建過去問**（令和元年〜令和7年）を保存するフォルダ。年度はファイル名で区別する（例：`令和3年_問題.pdf` / `令和3年_正解.pdf`）。

## ⚠️ 重要：公開リポジトリにコミットしない

- 過去問PDF等は RETIO 等の**著作権物**。このリポジトリは GitHub Pages で公開しているため、**問題・正解のファイルは push しない**。
- `.gitignore` で `*.pdf / *.jpg / *.png / *.zip` 等を除外済み。**実物ファイルはこのフォルダにローカル保存するだけ**でよい（git は無視する）。
- 追跡されるのは、この README と自作の**採点メモ（.md）**のみ。

## 使い方（実物過去問の運用・2026-07-19合意）

1. **日曜（2.5h）に自力で50問・2時間・時間厳守**で解く（本番の解き順で）。
2. 解答中、各問に**確信度マーク**：**無印＝自信あり／△＝消去法・あやしい／?＝勘**。
3. 自己採点し、**点数＋分野別＋「✗（誤答）＋△/?（あやしい正解）の問番号」**を Claude に共有。
4. Claude が論点解説＋e-Gov検証＋**法改正フィルタ**（古い年度で正解が変わる肢を警告）。誤答論点は自作○×に抽象化して exam_v6 バンクへ。
5. **復習対象＝誤答＋あやしい正解**（まぐれ正解を炙り出す）。理解の確認＝**4肢すべての根拠を言えるか**。

## 年度の割り当て（案）

- **R2（7〜8月）**：平成30・令和元・2・3・4 を運用力づくりに。
- **R3（9月・直前）**：**令和5・6（＋7）は温存**し、直前期に最新年で仕上げる。

（採点メモは `令和X年_採点.md` 等で残すと、Claude が誤答論点を拾いやすい。）
