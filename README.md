# 宅建士 合格ナレッジ・ポータル

不動産会社を運営するために**宅地建物取引士**の資格取得を目指す学習プロジェクト。

## 概要

試験合格に向けた重要ポイントをまとめたインタラクティブな学習ポータルサイトです。
試験の出題傾向を分析し、頻出事項を視覚的に整理しています。

## ドキュメント

| ファイル | 内容 |
|---------|------|
| [`exam_v6.html`](exam_v6.html) | インタラクティブ学習ポータル（ブラウザで開くだけ） |
| [`takken_guide_outline.md`](takken_guide_outline.md) | 試験対策ガイド アウトライン（全4科目の重要ポイント一覧） |

## 学習コンテンツ

| 科目 | 主な内容 |
|------|---------|
| **宅建業法** | 免許制度・宅地建物取引士、35条（重要事項説明）、自ら売主制限（8種制限）、媒介契約・報酬計算、クーリングオフ |
| **権利関係** | 民法・借地借家法・区分所有法・不動産登記法 |
| **法令上の制限** | 都市計画法・建築基準法・農地法・土地区画整理法 |
| **税・価格・その他** | 不動産取得税・固定資産税・印紙税・鑑定評価 |

## 機能

- **知識カード** — 分野別に色分けされたカードで重要事項を整理
- **比較表** — 35条 vs 37条、媒介契約3類型など混同しやすい項目を一覧化
- **一問一答クイズ** — 宅建業法の頻出問題12問（正答率バー付き）
- **計算例ガイド** — 報酬計算・手付金保全措置の判定フローを図解

## 使い方

`exam_v6.html` をブラウザで開くだけで動作します。インストール・サーバー不要。

```bash
open exam_v6.html   # macOS
start exam_v6.html  # Windows
```

## 資格について

**宅地建物取引士（宅建士）** は不動産取引における唯一の国家資格です。

- 毎年10月第3日曜日に実施
- 合格率 約15〜17%
- 独占業務：重要事項の説明（35条書面）、37条書面への記名
