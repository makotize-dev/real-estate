## 2.1 宅建士の日常業務の全体像

### 2.1.1 業務の全体フロー

不動産取引における宅建士の業務は、顧客との最初の接点から契約完了、そしてアフターフォローまで、一連のプロセスで構成されています。

**【不動産売買取引の全体フロー】**

```
①顧客対応・ニーズヒアリング
    ↓
②物件調査・資料収集
    ↓
③物件案内・内覧対応
    ↓
④価格交渉・条件調整
    ↓
⑤重要事項説明（独占業務）
    ↓
⑥契約締結・37条書面交付（独占業務）
    ↓
⑦決済・引渡しサポート
    ↓
⑧アフターフォロー
```

### 2.1.2 独占業務と非独占業務の区分

| 業務カテゴリ | 独占業務 | 主な業務内容 | 宅建士以外の従事可否 |
|------------|---------|------------|-------------------|
| 重要事項説明 | ○ | 35条書面による説明 | 不可 |
| 35条書面への記名押印 | ○ | 重要事項説明書への署名 | 不可 |
| 37条書面への記名押印 | ○ | 契約書への署名 | 不可 |
| 顧客対応 | × | 問い合わせ対応、ニーズヒアリング | 可 |
| 物件調査 | × | 登記簿・現地調査 | 可 |
| 物件案内 | × | 内覧対応 | 可 |
| 価格交渉 | × | 条件調整 | 可 |
| 広告作成 | × | チラシ・Web掲載 | 可 |
| 契約書作成 | × | 書類準備 | 可 |

この区分から分かるように、**実際の業務時間の大半は非独占業務**であり、独占業務は取引の重要な節目で実施される「最終確認・責任明確化」の役割を担っています。

## 2.2 独占業務の実務詳解

### 2.2.1 重要事項説明の実際の流れ

**【標準的な重要事項説明の所要時間】**

- **一般的な戸建て・マンション売買**：30分～60分
- **複雑な法令制限がある物件**：60分～90分
- **投資用物件（収益物件）**：60分～120分
- **賃貸借契約**：15分～30分

**【重要事項説明の実施手順】**

**STEP 1：事前準備（説明の1週間～3日前）**

1. 物件調査の実施
   - 登記簿謄本の取得と確認
   - 公図・測量図の取得
   - 都市計画図の確認
   - 建築確認済証・検査済証の確認
   - 上下水道・ガス等インフラの確認
   - 埋蔵文化財包蔵地の確認
   - 水害ハザードマップの確認

2. 35条書面の作成
   - 調査結果の整理
   - 専門用語の平易な説明文作成
   - 図面・資料の準備
   - 売主への確認事項の整理

3. 事前送付（推奨）
   - 買主に35条書面を事前送付
   - 質問事項の事前受付
   - 説明日時の調整

**STEP 2：当日の説明（契約日の直前または同日）**

1. 宅建士証の提示（法定義務）
   - 顔写真付き宅建士証を買主に提示
   - 宅建士であることの証明

2. 35条書面の読み合わせ
   - 項目ごとに丁寧に説明
   - 専門用語は平易な言葉で補足
   - 図面を使った視覚的説明
   - 買主の理解度を確認しながら進行

3. 質疑応答
   - 買主からの質問に回答
   - 不明点の解消
   - 追加資料の提示

4. 署名・捺印
   - 買主の署名・捺印
   - 宅建士の記名押印
   - 書面の交付

**STEP 3：記録保管**

- 説明済み35条書面の保管（5年間）
- 説明経過の記録
- 質疑応答内容のメモ保管

### 2.2.2 重要事項説明で特に注意すべき項目

**【税理士が関与する際の強み発揮ポイント】**

1. **固定資産税・都市計画税の説明**
   - 年税額の確認
   - 引渡し日での日割り精算計算
   - 評価替えのタイミング説明
   
   **計算例：**
   ```
   年間固定資産税：180,000円
   都市計画税：36,000円
   合計：216,000円
   
   引渡し日：7月1日
   売主負担：1月1日～6月30日（181日分）
   買主負担：7月1日～12月31日（184日分）
   
   売主負担額：216,000円 × 181日 / 365日 = 107,080円
   買主負担額：216,000円 × 184日 / 365日 = 108,920円
   ```

2. **登録免許税の説明**
   - 所有権移転登記の税額
   - 抵当権設定登記の税額
   - 軽減措置の適用可否
   
   **税額例（5,000万円の居住用物件）：**
   ```
   所有権移転登記（原則）：5,000万円 × 2.0% = 100万円
   所有権移転登記（軽減）：5,000万円 × 0.3% = 15万円
   
   軽減適用条件：
   ・自己居住用
   ・床面積50㎡以上
   ・築年数要件（耐震基準適合）
   ```

3. **取得後の税務上の取扱い説明**
   - 不動産取得税（取得後6ヶ月～1年半後に課税）
   - 減価償却の計算方法
   - 譲渡所得税の将来試算
   
   これらは税理士の専門領域であり、**他の不動産業者との明確な差別化ポイント**となります。

### 2.2.3 契約書（37条書面）作成の実務

**【37条書面の記載必須事項】**

1. **当事者の表示**
   - 売主・買主の氏名・住所
   - 法人の場合は代表者氏名

2. **物件の表示**
   - 所在地・地番
   - 地目・地積（土地）
   - 家屋番号・種類・構造・床面積（建物）

3. **代金・支払条件**
   - 売買代金額
   - 手付金額
   - 中間金額と支払時期
   - 残代金額と支払時期
   - 支払方法

4. **所有権移転・引渡しの時期**
   - 具体的な年月日

5. **契約不適合責任**
   - 責任の内容
   - 期間の定め
   - 免責事項

**【税理士が作成に関与する際の付加価値】**

- 税務上有利な引渡し時期の提案（年度またぎの検討）
- 減価償却開始時期を考慮した引渡し日設定
- 親族間売買の場合の時価との乖離リスク説明
- 消費税の取扱い（個人・法人、課税事業者の判定）

## 2.3 非独占業務の実務詳解

### 2.3.1 物件調査業務の詳細

物件調査は重要事項説明の基礎となる重要業務です。調査漏れは後日の紛争原因となるため、チェックリストを用いた網羅的な調査が必須です。

**【物件調査チェックリスト】**

**□ 登記情報の確認**
- 登記簿謄本（全部事項証明書）取得
- 所有者の確認
- 抵当権等の担保権の確認
- 差押え・仮差押えの有無
- 地目・地積・床面積の確認

**□ 公図・測量図の確認**
- 公図の取得
- 隣地との境界確認
- 測量図の有無と内容確認
- 境界標の現地確認

**□ 法令制限の調査**
- 都市計画法：用途地域、建蔽率、容積率
- 建築基準法：接道義務、建築制限
- その他の法令：農地法、文化財保護法など
- 自治体独自の条例

**□ インフラ調査**
- 上水道：公営水道か井戸か
- 下水道：公共下水道か浄化槽か
- ガス：都市ガスかプロパンか
- 電気：引込状況

**□ 災害リスク調査**
- 水害ハザードマップ
- 土砂災害警戒区域
- 津波浸水想定区域
- 活断層の有無

**□ 周辺環境調査**
- 嫌悪施設の有無
- 交通騒音・振動
- 日照・眺望阻害要因
- 将来の開発計画

**□ 建物調査（中古の場合）**
- 建築年月日
- 建築確認済証・検査済証の有無
- 増改築履歴
- 耐震診断結果
- 修繕履歴
- 設備の状況

**調査にかかる標準的な費用と時間**

| 調査項目 | 費用 | 所要時間 |
|---------|------|---------|
| 登記情報 | 1,000円/通 | 即時～1日 |
| 公図・測量図 | 500円/通 | 即時～1日 |
| 都市計画図 | 無料～500円 | 即時～1日 |
| 現地調査 | 交通費のみ | 1～3時間 |
| 役所調査 | 無料 | 2～4時間 |
| **合計** | **3,000円程度** | **1～2日** |

### 2.3.2 価格査定業務

**【価格査定の3つの手法】**

1. **取引事例比較法**
   - 近隣の類似物件の成約事例から査定
   - 最も一般的な方法
   - データベース（レインズ等）の活用

   **査定式：**
   ```
   査定価格 = 事例価格 × 面積比 × 立地補正 × 築年数補正 × 個別条件補正
   
   例：
   事例：駅徒歩10分、築10年、80㎡、3,000万円
   対象：駅徒歩15分、築12年、75㎡
   
   3,000万円 × (75/80) × 0.95（立地） × 0.98（築年） = 2,650万円
   ```

2. **収益還元法**
   - 投資用物件の査定に使用
   - 想定家賃収入から逆算
   
   **査定式：**
   ```
   査定価格 = 年間純収益 / 還元利回り
   
   例：
   年間家賃収入：600万円
   経費率：20%
   年間純収益：480万円
   還元利回り：6%
   
   査定価格 = 480万円 / 0.06 = 8,000万円
   ```

3. **原価法**
   - 再調達価格から減価償却
   - 新築・特殊物件の査定に使用

**【税理士の強み発揮ポイント】**

投資用物件の査定において、税理士は以下の追加分析を提供できます：

- **税引後キャッシュフロー分析**
  ```
  税引前CF：200万円
  所得税（税率33%）：30万円
  税引後CF：170万円
  実質利回り = 170万円 / 8,000万円 = 2.125%
  ```

- **減価償却による節税効果の数値化**
- **法人化した場合のシミュレーション**
- **出口戦略（売却時）の譲渡所得税試算**

### 2.4.2 税理士事務所における宅建士の働き方

**【パターン1：税理士本人が宅建士資格を保有】**

- **業務配分**：税務業務70% + 不動産業務30%
- **メリット**：追加人件費不要、顧客との信頼関係活用
- **デメリット**：専任性の確保が課題
- **適した規模**：個人事務所～小規模事務所

**【パターン2：専任宅建士を雇用】**

- **業務配分**：不動産業務専従
- **メリット**：専任性確保、業務拡大可能
- **デメリット**：人件費負担（年間400～600万円）
- **適した規模**：中規模～大規模事務所

**【パターン3：税理士+専任宅建士のハイブリッド】**

- **業務配分**：税理士がコンサル、専任宅建士が実務
- **メリット**：最も効率的な役割分担
- **デメリット**：組織マネジメントが必要
- **適した規模**：中規模以上

## 2.5 税理士事務所が不動産業務を行う際の実務上の注意点

### 2.5.1 事務所要件の確保

宅建業免許取得には、以下の事務所要件を満たす必要があります：

**【事務所の物理的要件】**

1. **独立性の確保**
   - 専用の出入口（税理士業務と共用可の場合も）
   - 固定式パーティション等での区画
   - 「宅地建物取引業○○事務所」の標識設置

2. **必要な設備**
   - 応接セット（商談スペース）
   - デスク・椅子
   - 電話・FAX
   - パソコン
   - 書類保管キャビネット

3. **面積の目安**
   - 最低10㎡程度
   - 応接セットを配置できるスペース

**既存の税理士事務所に不動産部門を追加する場合の工夫**

```
【レイアウト例：30㎡の事務所の場合】

│  ・書庫                  │
├─────────────────────┤
│  パーティション（固定式）    │
├─────────────────────┤
│  不動産業務エリア（15㎡）   │
│  ・応接セット            │
│  ・物件資料棚            │

```

### 2.5.2 帳簿書類の管理

宅建業法により、以下の帳簿書類の作成・保管が義務付けられています：

**【法定帳簿】**

1. **業務に関する帳簿**
   - 取引年月日
   - 取引物件の所在・種別
   - 取引当事者の氏名・住所
   - 取引金額
   - 保管期間：取引完了後5年間

2. **従業者名簿**
   - 氏名・住所・生年月日
   - 宅建士証番号（宅建士の場合）
   - 保管期間：退職後10年間

**税理士業務との親和性**

税理士は帳簿記録・保管に慣れているため、この要件は問題なく対応できます。むしろ、**税務会計ソフトと連動させることで、不動産取引の収益管理と税務申告を一体化**できる強みがあります。

### 2.5.3 報酬の明示義務

宅建業法により、報酬額を事務所に掲示する義務があります：

**【報酬の法定上限】**

**売買・交換の場合**

| 物件価格 | 報酬上限（税抜） |
|---------|----------------|
| 200万円以下 | 物件価格×5% |
| 200万円超～400万円以下 | 物件価格×4% + 2万円 |
| 400万円超 | 物件価格×3% + 6万円 |

**報酬計算例**

```
物件価格3,000万円の場合：
3,000万円 × 3% + 6万円 = 96万円
消費税10%：9.6万円
合計：105.6万円
```

**低廉な空き家等の特例**

400万円以下の物件で、調査費用が通常より多くかかる場合、最大18万円＋消費税（合計19.8万円）まで報酬を受け取れる特例があります。

**税理士事務所の場合の報酬設定**

一般の不動産会社と異なり、以下のような複合的な報酬体系が可能です：

```
【例：相続対策としての不動産購入支援】

①不動産仲介手数料：105.6万円（3,000万円物件）
②相続税シミュレーション：30万円
③不動産法人設立支援：50万円
④設立後の税務顧問：年間60万円

初年度合計：245.6万円
2年目以降：60万円/年
```

**【法律知識】**

### 2.6.3 継続的な学習の必要性

**【法改正への対応】**

不動産関連法令は頻繁に改正されます。主な改正例：

- **令和2年（2020年）**：水害リスク説明の義務化
- **令和3年（2021年）**：低廉な空き家等の特例創設
- **令和4年（2022年）**：所有者不明土地関連法制定

**【継続的学習の方法】**

1. **法定講習（5年ごと）**
   - 宅建士証更新時に受講義務
   - 最新の法令・判例を学習

2. **業界団体の研修**
   - 全国宅地建物取引業協会連合会（全宅連）
   - 不動産流通経営協会（FRK）

3. **税理士会の不動産研修**
   - 税理士向けの不動産実務研修
   - 相続対策セミナー

4. **専門書・実務書の読破**
   - 月刊誌「税理」「税経通信」の不動産特集
   - 「不動産取引の税務」などの専門書

## 2.7 まとめ：第2章のポイント

### 宅建士の実務の全体像

✓ 独占業務は取引の重要な節目で実施される責任明確化の役割
✓ 日常業務の大半は調査・交渉・事務処理などの非独占業務
✓ 重要事項説明は30～120分かかる重要業務
✓ 物件調査は取引の基礎となる最重要業務

✓ 既存顧客への追加サービスとして展開
✓ 平日中心の業務で土日休みが可能
✓ 税務知識を活かした高付加価値サービス
✓ 1案件あたりの報酬が高く、件数は限定的

### 税理士の差別化ポイント

✓ 税引後キャッシュフローでの投資分析
✓ 相続税・譲渡所得税を含めた総合提案
