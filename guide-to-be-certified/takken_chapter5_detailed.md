## 5.1 学習時間と開始時期の設定

### 5.1.1 必要学習時間の算定

**【一般受験生との比較】**

| 項目 | 一般受験生 | 税理士 | 削減率 |
|------|-----------|--------|--------|
| 宅建業法 | 120時間 | 120時間 | 0% |
| 権利関係 | 160時間 | 100時間 | ▲37.5% |
| 法令上の制限 | 80時間 | 80時間 | 0% |
| 税・その他 | 40時間 | 20時間 | ▲50% |
| **合計** | **400時間** | **320時間** | **▲20%** |

**税理士が学習時間を短縮できる理由**

1. **権利関係（民法）の基礎知識**
   - 税理士試験で民法の基本を学習済み
   - 契約、債権、物権の概念を理解している
   - 相続分野は実務経験あり
   - 判例の読み方に慣れている

2. **税法の専門知識**
   - 不動産取得税、固定資産税は既知
   - 譲渡所得、減価償却の計算が得意
   - 登録免許税、印紙税も理解済み
   - 税法5問は復習程度で満点狙える

3. **法律学習のスキル**
   - 条文の読解力が高い
   - 「ただし書き」「みなす」等の法律用語に慣熟
   - 論理的思考力が訓練されている
   - 過去問分析の方法を体得している

**税理士業務との両立ポイント**


**3ヶ月コースの特徴**

✓ メリット：
- 記憶が新鮮なまま試験に臨める
- 学習期間が短いためモチベーション維持しやすい
- 集中力が高まる

✗ デメリット：
- 1日の学習時間が長く、疲労蓄積
- 仕事との両立が困難

- 体調不良時のリカバリーが困難

**税理士へのアドバイス**

既に資格試験の学習経験があり、効率的な学習方法を知っている税理士でも、**6ヶ月コースを推奨**します。理由は以下の通りです：

1. 税理士業務との両立を考慮
2. 理解を深めることで応用問題に対応
3. 体調管理やトラブル時の余裕確保
4. 繰り返し学習による定着率向上

## 5.3 科目別攻略法

### 5.3.1 宅建業法（20問）の完全攻略

**【重要度：最重要★★★★★】**

**学習時間：120時間**

**得点目標：18～20点（90～100%）**

**科目の特徴**

- 50問中20問（40%）を占める最重要科目
- 実務に直結した内容
- 暗記中心で得点しやすい
- ほぼ毎年同じテーマから出題
- 法改正の影響を受けやすい

**学習の進め方**

```
【STEP 1：全体像の把握（20時間）】

テキストを通読：
- 宅建業の定義・免許制度
- 宅建士制度
- 営業保証金・保証協会
- 業務上の規制
- 35条・37条
- 報酬
- 監督・罰則
- 住宅瑕疵担保履行法

重要度の確認：
★★★★★：35条、37条、報酬
★★★★☆：業務上の規制、宅建士制度
★★★☆☆：免許制度、保証協会
★★☆☆☆：監督・罰則

【STEP 2：重点分野の深掘り（40時間）】

35条（重要事項説明）：
- 説明が必要な事項を完全暗記
- 売買・賃貸借での違いを整理
- 宅建士証提示義務
- IT重説の要件

37条（契約書面）：
- 記載事項を完全暗記
- 35条との違いを明確化
- 記名押印の主体

報酬：
- 計算式を完全マスター
- 低廉な空き家等の特例
- 媒介・代理での違い

【STEP 3：過去問演習（40時間）】

10年分×20問＝200問を3周：
- 1周目：全問解く、間違いを確認
- 2周目：間違えた問題のみ解く
- 3周目：全問を制限時間内（20分）で解く

【STEP 4：仕上げ（20時間）】

- 最頻出テーマの総復習
- 法改正部分の確認
- 予想問題演習
```

**暗記のコツ**

```
【数字の暗記】

「5」が基本：
- 事務所：従業者5人に1人の宅建士
- 案内所：案内所にも専任宅建士
- 免許の有効期間：5年
- 宅建士証の有効期間：5年
- 帳簿の保存期間：5年

「10」も重要：
- 従業者名簿の保存：退職後10年
- 広告の開始時期：許可・確認後

【語呂合わせ活用】

報酬の計算：
「さぶろく・よんはち・ごいちぜろ」
200万円以下：5%＋6万円
400万円以下：4%＋8万円
400万円超：3%＋10万円
（実際は3%+6万円だが、覚えやすさ優先）
```

**税理士の強みを活かすポイント**

1. **契約書の理解**
   - 35条・37条は契約書作成の経験が活きる
   - 「記載事項」と「説明事項」の違いが明確に理解できる

2. **報酬計算**
   - 計算問題は税理士の得意分野
   - 消費税の取扱いも即座に理解できる
