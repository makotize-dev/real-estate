## 4.1 試験実施機関と制度の基本

### 4.1.1 試験実施機関

**【一般財団法人 不動産適正取引推進機構】**

- **正式名称**：一般財団法人 不動産適正取引推進機構
- **略称**：RETIO（レティオ）
- **公式サイト**：https://www.retio.or.jp/
- **所在地**：〒105-0001 東京都港区虎ノ門3-8-21 第33森ビル3階
- **電話**：03-3435-8181（試験部）

**組織の役割**

1. **宅建士試験の実施**
   - 試験問題の作成
   - 試験会場の設営
   - 採点・合格発表
   - 統計データの公表

2. **不動産取引の適正化推進**
   - 紛争事例の収集・分析
   - 調査研究
   - 情報提供

3. **研修・講習の実施**
   - 登録実務講習
   - 法定講習

### 4.1.2 試験の法的根拠

**宅地建物取引業法 第16条の2**

```
第16条の2（宅地建物取引士資格試験）

国土交通大臣は、宅地建物取引士資格試験を行う。

2 前項の試験は、宅地建物取引業に関し必要な知識について行う。

3 国土交通大臣は、第1項の試験の実施に関する事務を、
国土交通省令で定めるところにより、一般財団法人不動産
適正取引推進機構に委任することができる。
```

この規定に基づき、国土交通大臣から委任を受けた不動産適正取引推進機構が実際の試験実施を担当しています。

### 4.2.2 申込方法の詳細

**【推奨：インターネット申込】**

**メリット**
- 24時間申込可能（締切日は21:59まで）
- 郵送より申込期間が長い（7/1～7/31）
- 受験料の支払方法が多様（クレジットカード、コンビニ決済等）
- 申込完了の確認がその場でできる

**手順**

```
STEP 1：顔写真データの準備
- JPEGまたはPNG形式
- ファイルサイズ：5MB以内
- 画像サイズ：横400×縦500ピクセル以上
- 6ヶ月以内に撮影したもの
- 正面、無帽、無背景

STEP 2：不動産適正取引推進機構サイトへアクセス
- https://www.retio.or.jp/
- 「宅建士試験」→「インターネット申込」

STEP 3：必要事項の入力
- 氏名、住所、生年月日
- 受験地の選択（住所地以外でも可）
- 登録講習修了者は修了番号入力

STEP 4：顔写真のアップロード

STEP 5：受験料の支払い
- クレジットカード
- コンビニ決済
- ペイジー

STEP 6：申込完了メールの確認
- 受付番号を必ず保存
```

**【郵送申込】**

**手順**

```
STEP 1：試験案内の入手
- 各都道府県の試験会場で配布
- または郵送請求（返信用封筒＋切手140円同封）

STEP 2：受験願書の記入
- 黒のボールペンで丁寧に記入
- 訂正は二重線＋訂正印

STEP 3：顔写真の貼付
- 縦4.5cm×横3.5cm
- 6ヶ月以内に撮影
- 裏面に氏名記入

STEP 4：受験料の納付
- 8,200円分の証紙を購入
- 願書の所定欄に貼付

STEP 5：郵送
- 簡易書留で送付
- 締切日（7/15）の消印有効
```

**申込時の注意点**

| 項目 | 注意事項 |
|------|---------|
| 氏名 | 戸籍に記載された正式な氏名 |
| 住所 | 合格証書送付先になる |
| 受験地 | 住所地以外でも選択可能 |
| 登録講習 | 修了者は必ず修了番号を記入 |
| 顔写真 | 不鮮明な写真は受験不可になる可能性 |

### 4.2.4 受験地の選択

**【全国47都道府県で実施】**

- **原則**：住所地の都道府県で受験
- **例外**：住所地以外での受験も可能

```
【ケース1：東京在住、地方出張が多い】

【ケース2：地方在住、都市部の方が会場が多い】
→ 最寄りの都市部で受験（会場選択肢が多い）

【ケース3：試験日前後に出張予定】
→ 出張先の都道府県で受験（効率化）
```

## 4.3 試験当日の詳細

### 4.3.1 試験会場と座席

**【試験会場】**

- 大学、専門学校、公共施設など
- 受験票に記載された会場以外での受験は不可
- 会場の変更は原則不可

**【座席】**

- 受験番号順に指定（選択不可）
- 教室形式（机・椅子は学校の備品）
- 隣との間隔：約50cm程度

**会場の事前確認（推奨）**

```
試験日の1週間前～前日に：
□ 会場までの経路確認
□ 所要時間の実測
□ 最寄り駅からの徒歩ルート
□ 昼食場所の確認（試験は午後）
□ トイレの位置確認
```

### 4.3.2 試験当日のタイムスケジュール

**【標準的な当日スケジュール】**

```
11:30  自宅出発（余裕を持って）
12:15  試験会場到着
12:30  受験票・身分証明書の確認
       トイレ、最終確認
12:50  試験室入室開始
13:00  試験開始
       - 注意事項の説明（5分程度）
       - 問題配布
       - 解答開始
14:00  途中退出可能時間帯開始
       （ただし推奨しない）
15:00  試験終了
       - 解答用紙回収
       - 問題用紙は持ち帰り可
15:10  退室
15:30  自己採点（予備校の速報利用）
```

### 4.3.3 持ち物チェックリスト

**【必須の持ち物】**

✓ **受験票**
  - 忘れた場合は試験会場で再発行（写真持参必要）
  - 汚損・紛失の場合も同様

✓ **本人確認書類（顔写真付き）**
  - 運転免許証
  - パスポート
  - マイナンバーカード
  - 住民基本台帳カード（顔写真付き）
  
✓ **筆記用具**
  - HBまたはBの鉛筆（シャープペンシル可）
  - 消しゴム
  - 鉛筆削り（推奨）

✓ **腕時計**
  - スマートウォッチ等の通信機能付きは不可
  - 計算機能付きも不可
  - アナログ・デジタルは問わない

**【推奨の持ち物】**

□ 上着（空調対策）
□ 飲み物（透明な容器、試験中は机の下）
□ ハンカチ・ティッシュ
□ 目薬（疲れ目対策）
□ 糖分補給（チョコレート等、休憩時に）
□ 軽食（試験前の昼食用）

**【持ち込み禁止物】**

✗ 携帯電話・スマートフォン（電源OFF、鞄へ）
✗ 参考書・ノート類
✗ 計算機
✗ 筆箱（机上に置けるのは筆記用具のみ）
✗ ペットボトル飲料（机上不可、机の下ならOK）

### 4.3.4 試験中の注意事項

**【解答方法】**

- マークシート方式（塗りつぶし型）
- HBまたはBの鉛筆で丁寧に塗る
- 薄い場合は読み取りエラーの可能性
- 消し残しも誤答扱い

**【途中退出】**

- 試験開始後1時間（14:00）から可能
- 途中退出後の再入室は不可
- 解答用紙提出、問題用紙は持ち帰り可

**税理士へのアドバイス**

```
途中退出は原則しない：
- 2時間しかないため、見直しに時間を使う
- マークミスの確認
- 迷った問題の再検討
- 全問解答しているか再確認

税理士試験と同様の精神で：
- 最後まで諦めない
- ケアレスミスを徹底排除
- 時間配分を守る
```

**出題例（重要事項説明）**

```
問：宅地建物取引業者が行う宅地の売買の媒介に
係る重要事項の説明に関する次の記述のうち、
宅地建物取引業法の規定によれば、正しいものは
```

**出題例（相続）**

```
問：相続に関する次の記述のうち、民法の規定に
よれば、誤っているものはどれか。

1. 相続人が数人あるときは、相続財産は、その共有に
   属する。

2. 被相続人の子が相続の開始以前に死亡したときは、
   その者の子がこれを代襲して相続人となる。

3. 相続人が被相続人の配偶者及び子である場合、
   配偶者の相続分は2分の1である。

4. 遺留分を有する相続人が、被相続人の生前に
   遺留分を放棄しようとする場合、家庭裁判所の
   許可を得る必要はない。

正解：4（遺留分の放棄は家裁の許可必要）
```

**税理士の強み**

- 民法の基礎知識（契約、債権、相続）は税理士試験で学習済み
- 相続分野は実務経験があり、圧倒的に有利
- ただし、借地借家法・区分所有法は新規学習が必要

**税理士の圧倒的優位性**

- 税法5問は既知の内容がほとんど
- 実質的に「ボーナス問題」
- 一般受験生は苦戦するが、税理士は満点狙える
- この分野での貯金が合格を確実にする

## 4.5 合格基準と合格率の詳細分析

### 4.5.1 相対評価制度の仕組み

**【合格点の決定プロセス】**

```
STEP 1：試験実施

STEP 2：全受験者の得点分布を集計

        （絶対的な基準ではなく、相対的に決定）

STEP 4：合格点を公表
```

**過去10年間の合格点推移**

| 年度 | 合格点 | 受験者数 | 合格者数 | 合格率 |
|------|--------|---------|---------|--------|
| 2024年 | 37点 | 243,574人 | 45,344人 | 18.6% |
| 2023年 | 36点 | 246,393人 | 44,047人 | 17.9% |
| 2022年 | 36点 | 246,018人 | 38,568人 | 15.7% |
| 2021年 | 34点 | 242,691人 | 41,471人 | 17.1% |

| 2018年 | 37点 | 265,444人 | 33,360人 | 12.6% |

| 2016年 | 35点 | 245,742人 | 30,589人 | 12.4% |

**データ分析**

1. **合格点のレンジ**：31～37点（50点満点）
2. **平均合格点**：35.2点（約70%）
3. **合格率の変動**：12.3%～18.6%
4. **2021年以降の傾向**：合格率が上昇（15～18%台）

**合格点変動の要因**

- 問題の難易度（特に権利関係）
- 法改正の影響
- 社会情勢（不動産市場の活況・不況）
- 人材需要（法定配置要件による需要）

### 4.5.2 登録講習修了者の優遇措置

**【5問免除制度】**

**対象者**

- 宅建業に従事している者
- 登録講習（国土交通大臣登録講習）を修了した者
- 修了証の有効期限内（修了日から3年以内）の者

**免除内容**

問46～問50の5問が免除され、自動的に5点が加算される

| 問題 | 内容 | 難易度 |
|------|------|--------|
| 問46 | 住宅金融支援機構法 | 普通 |
| 問47 | 景品表示法 | やや難 |
| 問48 | 統計 | 普通 |
| 問49 | 土地の知識 | 普通 |
| 問50 | 建物の知識 | 普通 |

**税理士にとっての登録講習の必要性**

```
【不要と判断する理由】

①税理士は税法で確実に5点取得できる
  → 5問免除の優位性が低い

②受講費用と時間がかかる
  → 費用：約2万円
  → 期間：約2ヶ月
  → 通学：2日間

③宅建業に従事していない
  → 受講資格がない（税理士事務所勤務では不可）

【結論】
税理士は登録講習を受講せず、
50問全問解答で合格を目指すべき
```

### 4.5.3 統計問題の対策

**【問48：統計問題】**

試験年度の直近の統計データから出題されます。

**主な出題データ**

1. **地価公示**（国土交通省）
   - 全国平均の変動率
   - 用途別（住宅地、商業地）の動向

2. **建築着工統計**（国土交通省）
   - 新設住宅着工戸数
   - 持家・貸家・分譲住宅の内訳

3. **法人企業統計**（財務省）
   - 不動産業の売上高
   - 経常利益

4. **土地白書**（国土交通省）
   - 土地取引件数
   - 土地所有の状況

**対策方法**

```
【試験1ヶ月前（9月）に集中学習】

①予備校の統計資料集を入手
  - TACやLEC等が無料配布

②主要数値を暗記
  - 上昇・下落の傾向のみ把握
  - 細かい数値は不要

③過去問で出題パターン確認

所要時間：2～3時間程度で1点確保
```
